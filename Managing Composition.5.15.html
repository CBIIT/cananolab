Defining Nanomaterial Composing Elements
This section opens automatically when you open the Nanomaterial Entity section. See Adding Nanomaterial Entity Composition Annotations.
To define composing elements for the nanomaterial entity, follow these steps.
Click [Other] to enter an alternative to selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear
1.
Composing Element Type (required): Select from the drop-down list the type of composing element comprising this nanomaterial. Options include the following:
•
coat
•
core
•
lipid
•
modifier
•
monomer
•
repeat unit
•
shell
•
terminal group
2.
Chemical Name (required): Enter the chemical name given to the atomic or molecular structure that composes the center of the nanoparticle.
3.
PubChem Data Source/PubChem ID: Select one of the three PubChem databases: Compound, Substance, and BioAssay. The database is located here: http://pubchem.ncbi.nlm.nih.gov/. Enter the PubChem ID which is the PubChem identifier for a compound, a substance or a bioassay.
4.
Amount/Amount Unit: Select appropriate values and unit for this composing element information. Unit options are:
•
%
•
% mole
•
% vol
•
% wt/vol
5.
Molecular Formula Type: Select from the drop-down list the molecular formula type found in this biopolymer
•
SMARTS
•
SMILES
6.
Molecular Formula: Enter the chemical formula of the nanoparticle.
7.
Description: Enter a description for the composing element.
8.
Inherent Function: Inherent function is the characteristic behavior of a sample that results from the chemical and physical composition and properties of the entity.
Click the Add button to expand the page where you can enter this information.
a.
Function Type: Select from the drop-down list the function type category which best fits the nanomaterial entity.
•
Imaging
•
Targeting
•
Therapeutic
b.
Description: Enter any additional function information for the nanomaterial entity that the form does not already include.
c.
Click Save to add the inherent function details to the nanomaterial entity or Cancel to just close the window.
9.
Description: Enter any additional composing element information for the nanomaterialentity that the form does not already include.
10.
Click Save to add the composing element details to the sample.
After defining composing elements, continue the process described in Adding Nanomaterial Entity Composition Annotations.
