Displaying a Summary View of Leukocyte Proliferation In Vitro Characterizations
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Leukocyte Proliferation is an In Vitro Characterization.
If there are multiple In Vitro Characterizations added under a subcategory in the Navigation Tree, you can display a summary view of all the characterizations in one table. You can then print the table or save the table to a spreadsheet file.
To display all the characterizations in one table, follow these steps:
1.
Click Leukocyte Proliferation. The In Vitro Characterization - Leukocyte Proliferation page opens listing all the characterizations in one table, one characterization per row.
2.
For information about the the Summary View, see Characterization Summary View.
3.
