Submitting a Publication
This help topic opens from the Update Publication page, as well as from the Submit Report page. For more information about the Update Publication page, see Update Publication Page.
The Submit Publication feature allows you to identify a publication and associate it with one or more nanoparticles.
To define a nanoparticle publication, follow these steps:
1.
Click the Documents caNanoLab menu option. On the Manage Documents home page, click the Submit a New Publication link. This opens the Submit Publication page.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Asterisks indicate required fields.
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear

Enter the PubMed ID (click the Click to look up PubMed Identifier hypertext link where you can search for the ID). Once you enter the ID, the other appropriate data fields on this form, such as Title, Journal, Authors, etc., are automatically populated.
If you remove the PubMed ID, the related data fields are cleared.

Copy
This section replaces the Nanoparticle Sample Name field if you are submitting a publication from within an open nanoparticle. If that is the case, you can use this copy feature to associate the document with other nanoparticles to which you have access.
Select one or more nanoparticles to be associated with this publication.
Visibility
The term, Visibility, is used to describe the access to objects granted to logged in users. Select the group to which this publication is to be visible.
