Nanoparticles Overview
When you click the Nanoparticles option on the caNanoLab menu bar, the Manage Nanoparticle Home page appears. The Nanoparticle Links panel on the Manage Nanoparticle Home page provides the following options:
•
Submit a New Nanoparticle: This takes you to a form for creating a nanoparticle in the caNanoLab database. For more information, see Submitting a Nanoparticle Sample.
•
Search Existing Nanoparticles: This takes you to a form for launching a search for an existing nanoparticle. For more information, see Searching Existing Nanoparticles.
For more information about nanoparticles, see Description of a Nanoparticle.
