Adding a Solubility Physical Characterization
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Solubility is a Physical Characterization.
In the Navigation Tree the Add button next to the Solubility name enables you to add physical characterizations to caNanoLab.
If you do not have certain security privileges, the Add buttons will be hidden, and you will have read-only access to the existing items.
To record solubility data for a particle, follow these steps:
1.
On the Navigation Tree next to the Solubility heading click the Add button. The Physical Characterization-Solubility page opens.
2.
Complete the following fields on the form.
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point, your entry is sandwiched by [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear.
Solubility Characteristic

Summary
Characterization Source
Enter the source from which the characterization data is derived, such as a vendor or a laboratory (NCL).
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Navigation Tree. The asterisk indicates a required field.
Protocol Name - Version
Enter the protocol from which the data is derived and or enter identifying version information for the protocol from which the data is derived. A hyperlink of the protocol text file may appear. Click the hyperlink to open or save the file.
Description
Enter information not covered by other fields on the form.
Instrument Information
Instrument Type
Identify the kind of instrument used to obtain data.
Instrument Manufacturer
Specify the manufacturer of the instrument used to obtain the data.
Description
Enter information about instrumentation not covered by other fields.
Characterization File/Derived Data Information
Click the Add File/Derived Data link to extend the page where you can add characterization files consisting of graphs, spreadsheet or images, or derived data. For more information, see Loading Physical Characterization Files.
Note:
You can add as many files and derived data as you wish, or you can add derived data without adding files.

Derived Data

Name

•

Description
Enter an appropriate description for the derived data.
Copy to Other Nanoparticle Samples
If appropriate, this option copies files and summary information to one or more selected particles from the same vendor. Select one or more particle(s) to which the copied data is to be transferred. (Only appropriate particles are listed.) For more information, see Copying Characterizations.
Also Copy Derived Data?
Select this button to copy derived data, as well, to the selected particle.
3.
When you have finished, click Submit or click Reset to clear all fields on the form.
In the Navigation Tree, a hypertext link for the new characterization is added under Solubility.
