Searching Samples
This function is limited to users belonging to a specific research group. For more information, see Permissions and Visibility.
In the conventional sense, the term “sample” can refer to either of two entities in caNanoLab. For more information, see caNanoLab Sample Overview. This topic describes performing a search for a “Sample”. For information about searching for “Nanoparticles”, see Searching Nanoparticles.
To launch a search for a specific sample, follow these steps:
1.
Select Inventory > Search Samples. This opens the Search Samples page.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, in searching for sample NCL-8-234, you could enter NCL-8*.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab samples will be listed in the Search Results.
Sample Search Options
Description
Sample ID
Enter the fully qualified name of the Sample to be searched. Add an asterisk for a wild-card search.
Sample Type
Select from the drop-down list the name of the Sample Type being searched. Example: Dendrimer, Liposome.
Source
Select from the drop-down list the name of the Source from which the sample originated. Example: XYZ institute.
Source ID
Select from the drop-down list the unique identifier for each source.
Date Accessioned
Click the two calendars to define the range of dates during which the samples were accessioned in the lab.
Creator
Select from the drop-down list the name of the person who submitted the Sample.
Storage Location
Enter or select the location (Room, Freezer, Shelf, Box) in which this container is stored. Example: Freezer1, Box 12.
3.
Click Search to execute the search. To clear all fields, click the Reset button.
