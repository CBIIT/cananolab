Remote Report Search Results
If you click the Search button having defined no parameters, all of the nanoparticles located in the remote search display in the search results.
Once you have submitted a search for a Remote Report (Remote Reports Search), search results display in the caNanoLab screen. Results show pertinent report information and a URI to each report. Click the URI to locate and download a report.
Only public data can be retrieved from a remote search.
