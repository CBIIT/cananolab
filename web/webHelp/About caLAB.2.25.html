Uploading Input/Output Files
After you have created an Assay Run, caLAB allows you to upload input or output files of any type that are pertinent to the Assay Run. To do this, start with step 1 in this topic.
You can also add files to an existing Run that is already populated with data. To do this, start with step 4 in this topic.
Examples:
Input files
Instrument calibration data; paper describing a Sample in detail
Output files
Laboratory results; analyses of laboratory results
To upload one or more files, follow these steps:
1.
Immediately after creating an Assay run (Creating an Assay Run), with an Assay Run selected in the workflow tree, click either the Upload Input Files or Upload Output Files hypertext link in the right-hand panel of the browser.
OR
Select an existing Assay Run, click on the (+) symbols in front of the run, and select the In or Out folder. Click the Upload Files hypertext link to add files to the existing run.
This opens an applet where you can navigate to the file(s) to be uploaded. (You are not limited as to file type you can upload.)
2.
Using conventional selection techniques, select the file(s) in the left panel and move the file you select to right panel in the applet using the forward arrows (>>). A message opens indicating unacceptable characters for the file name. Click OK to proceed.
3.
Click the Upload Files button, and click OK to confirm the upload. Files are uploaded in a zipped format.
caLAB displays an Uploaded File Summary dialog box.
The Upload File applet still displays in the caLAB browser. You can stipulate to upload another file if you wish.
4.
Click Done to return to original page showing Upload... hypertext links. In the workflow tree, the Run now displays folder icons labeled In and Out. Clicking either of these links opens a new set of hypertext links in the right-hand panel of the browser:
–
Input: Use Aliquot, Upload Files, Download Files, and Mask Aliquot.
–
Output: Upload Files, Download Files, and Mask Aliquot
5.
To view a list of files uploaded into an Assay Run, click the Download Files but­ton. This opens a table displaying the Uploaded Input Files, with their caLAB-designated file name. For more information, see caLAB File Naming Conven­tions.
