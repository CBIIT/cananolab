Searching Protocols
A Protocol provides clear and detailed documentation about the precise actions taken iin creating or annotating a nanoparticle. See Managing Protocols.
To launch a search for a specific protocol, follow these steps:
1.
Select Search > Search Protocols. This opens the Search Protocols page.
2.
Enter text or select criteria as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, in searching for protocol ITA-8, you could enter ITA*.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
Note:
If you click the Search button having defined no parameters, all of the caNanoLab protocols will be listed in the Search Results.
Sample Search Options
Description
Protocol Type
Select from the drop-down list the name of the Protocol Type being searched.
Protocol Name
Enter the name of the protocol.
Protocol File Title
Enter the name of the protocol file title
3.
Click Search to launch the search. Click Reset to clear the search criteria.
