Managing caLAB Inventory
When you click the Inventory option on the menu bar, tasks associated with inventory management become available:
•
Managing Samples
•
Manage Aliquots
For more information, follow the topic hypertext links.
Note:
Each of these options allow you to “select an existing sample”. If you select either of those hyperlinks, a search form opens where you can initiate a search for an existing sample or aliquot. For more information, see Launching a caLAB Search.
