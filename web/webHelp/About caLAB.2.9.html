Working in caLAB
caLAB Workflow Overview
Workflows in caLAB must be incorporated into the system by the caLAB team; they become workflow templates. caLAB 1.0 has implemented one workflow: a reference workflow contributed by the Nanotechnology Characterization Laboratory (NCL). While the workflow is preconfigured and prepopulated with some data, you can add additional data and files to the workflow template.
After you have logged into caLAB, the application opens automatically to a Welcome page that displays a recommended laboratory workflow, represented in the following figure.
The caLAB online help table of contents also follows the same basic workflow.
A workflow can include any or all of the following tasks:
•
Create sample; performed by an administrator
•
Create aliquot; performed by an administrator
•
Execute a workflow
•
Create assay run/run
•
Upload input and output files for runs
•
Mask a file (input and output files)
•
Mask an aliquot
•
Search a workflow for run information; view search results
•
Search for samples and aliquots; view search results
For more information about each task in a workflow, click the appropriate menu option.
In this release, one workflow template, a reference inplementation for NCL, is implemented in the system.
Once you initiate tasks in caLAB 1.0, caLAB displays a “Run Tree" for a particular run that you prefer to work on. For example, when you create a new Assay Run, caLAB displays the selected Run in the left hand panel. When you search Runs, search results display in table format. From the table, you can choose a Run to view in detail, while a Run Tree displays in the left side panel.
If you select Workflow > Execute Workflow, two options display (described in Creating an Assay Run):
1.
Create a New Assay Run
2.
Select an Existing Assay Run.
Upon completing either of these tasks, the Assay Run displays in the workflow tree in the left-hand pane of the caLAB browser. The workflow displays pre-populated data. There are four assay types in this workflow--Prescreening, In Vitro, In Vivo and PCC. Each of the items listed under the assay types are assays. Only two assay types are populated in caLAB 1.0.
