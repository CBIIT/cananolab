Performing a Remote Reports Search
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle report results from the Login page.
The Remote Reports Search allows you to search remote caNanoLab nodes connected to the caGrid for available nanoparticle reports. Only publicly available reports are retrieved in this search.
To launch this search, follow these steps:
1.
Click the Reports caNanoLab menu option. On the Manage Report Home page, click the Search Existing Reports link. This opens the Search Report form.
2.
Click the Grid Search link at the top of the page.
3.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, caNanoLab searches all remote caNanoLab nodes for available nanoparticle reports.
Remote Report Search Fields
Description
Report Title
Enter the name of report.

Select from the drop-down menu the type of report being queried.
