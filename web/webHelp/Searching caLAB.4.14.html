Searching Nanoparticle Reports
The Search Report feature allows all caNanoLab users to find and view nanoparticle reports. This feature is accessible even to users for whom access to nanoparticles in caNanoLab is restricted.
To launch a search for a specific nanoparticle, follow these steps:
1.
Select Search > Search Reports, or for restricted users, select Search Characterization > Search Reports.
2.
On the Search Reports form that opens, complete one or more of the fields described in the following table. Add an asterisk for a wildcard search.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab reports will be listed in the Search Results.
