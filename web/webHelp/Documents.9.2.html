Documents Overview
When you click the Documents option on the caNanoLab menu bar, the Manage Documents home page appears. The Document Links panel on the page provides the following options:
•
Submit a New Publication: This takes you to a form for submitting a new caNanoLab publication. For more information, see Submitting a Publication.
•
Submit a New Report: This takes you to a form for creating a new caNanoLab report. For more information, see Submitting a Publication.
•
Search Existing Documents: This takes you to a form to launch a search for an existing publication or report of interest. For more information, see Searching Nanoparticle Documents.
Once a document has been submitted and associated with one or more specific nanoparticles, when the nanoparticle is opened in caNanoLab, the associated documents are listed along with the particle characterizations in the left sidebar. Where appropriate, the document list includes hypertext links so you can open the document(s).
If you work from the Documents menu option, you can associate multiple nanoparticles with a publication or report. If you add documents to an open nanoparticle, you can use the copy feature to associate the document with other nanoparticles.
