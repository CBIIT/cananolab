Size
Size, referring to physical magnitude, is one of the physical characteristics of a nanoparticle.
To record size data for a particle, follow these steps:
1.
On the Physical Characterization page, click the Size tab to open the form for recording nanoparticle size data.
2.
Complete the following fields on the form:
Size Characteristic
Description
Characterization Source
Enter the source from which the characterization data is derived, such as a vendor or a laboratory (NCL).
View Title
Enter a unique and meaningful name for the characterization. The title you enter will display on the Particle Tree.
Protocol Name
Enter the protocol from which the data is derived.
Protocol Version
Enter identifying version information for the protocol from which the data is derived.
Description
Enter information not covered by other fields on the form.
Instrument Information
 
Instrument Type
Select from the drop-down list the kind of instrument used to obtain data.
Other Instrument Type
Enter Instrument Type that is not listed in the drop-down list.
Instrument Manufacturer
Select the manufacturer of the instrument used to obtain the data.
Other Instrument Manufacturer
Enter Other Instrument Manufacturer that is not listed in the drop-down list.
Description
Enter information about instrumentation not covered by other fields.
Size Distribution
 
Number of Distributions
Enter the number of Size Distribution graphs for the particle
Number of Distributions
Note: If you enter any number greater than “0”, click the Update Distributions button at the bottom of the page. This extends the page with the corresponding number of additional fields so you can enter additional information for each unit.

Chemical Name
•
Description
Extended Page: Graph Data
 
Graph Type
Select from the drop-down list the appropriate graph type:
•

•

Characterization File Name
This field remains empty until you select the file to be uploaded. To do so, click the Load File button, opening the Load Characterization File page. For more information about completing the form, see Loading Characterization Files.
Average Mean
Enter the average value calculated from size measurements.
Z-Average
Enter the Z-average value calculated from size measurements.
Polydispersity Index (PDI)
PDI is a measure of the distribution of molecular weights in a given polymer sample. Enter the appropriate measurement.
3.
Once you have entered the appropriate information on the form, click Submit.
OR
4.
Click Reset to clear the fields on the form.
