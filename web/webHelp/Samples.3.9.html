Managing Aliquots
The Aliquot Links panel on the Manage Sample/Aliquot Home page provides the following options:
•
Creating a New Aliquot: This takes you to a form for entering information for a new sample you want to submit. For more information, see Creating A New Aliquot.
In caNanoLab, aliquots are named automatically when they are created. For more information, see Aliquot Naming Conventions.
•
Search Existing Aliquots: This takes you to a form for launching a search for an existing aliquot. For more information, see Searching Existing Aliquots.
