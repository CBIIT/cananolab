Managing Samples
Two options display when you select the Inventory > Manage Samples menu option in caNanoLab.
Creating a New Sample
This takes you to a form for entering information for a new sample you want to submit. For more information, see Creating a New Sample.
Select an Existing Sample
This takes you to a form for launching a search for an existing sample. For more information, see Searching Samples.
