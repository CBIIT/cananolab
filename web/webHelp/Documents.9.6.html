Nanoparticle Documents Search Results
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle document results from the Login page.
Nanoparticle Document Search Results display in the browser window in table format. Column headings are appropriate for the document(s) for which you searched; cell entries were defined when the report was submitted. Columns are sortable. Column headings are described in the following table.
If you click the Search button having defined no parameters, all of the caNanoLab nanoparticle documents will display in the search results.
Document Search Results Columns
Description
Title
Click the hypertext link to open the Report Page or Publication Page where you can view, edit, print or export details about the document.
Document Link
Click the hypertext link to open the document.
Document Category
The document category you selected when you created the report.
Research Category
The research category you selected when you created the document. (Publications only)
Associated Particle Sample Names
Names of nanoparticle(s) associated with the document
Create Date
Date the document was submitted
Location
Local or remote grid nodes where the report(s) was (were) located
For any hit, click on the hypertext link in the Report Title column (first) to open the associated report or file.
