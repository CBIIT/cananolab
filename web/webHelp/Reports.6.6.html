Report Page
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle report results from the Login page.
The Report File page opens when you click any of the hypertext file name links in the Nanoparticle Reports Search Results page.
The page that displays is the same Report File page on which the report was submitted into caNanoLab. Details are described in the following table. Asterisks indicate required fields.
Nanoparticle Report Columns
Description
Report File Type
The report type.
File Name
The name assigned by caNanoLab to the file.
File Title
The report title assigned when the report was submitted.
File Description
The description of the file.
Visibility
The group to which this report is visible.
