Understanding the Navigation Tree
Once you submit or search and select an existing nanoparticle with the Nanoparticles menu option (see Nanoparticles Overview), the Navigation Tree heading displays in the left-hand panel of caNanoLab with the selected “Viewing Nanoparticle [name]”. The Navigation Tree comprises categories which you can use to add annotations to the nanoparticle. Each category is a button, and when you click a button, options may expand below to show subcategories. Click the heading again, and the subcategories collapse.
The following list provides a general description of how you can use each category.
•
General Information: Display general information about the nanoparticle entered during the submission process.
•
Sample Management: Option currently not available.
•
Sample Composition: Add composition properties to the nanoparticle. For more information, see Sample Composition.
•
Physical Characterizations: Add distinctive characteristics or essential features of the nanoparticle determined through analytical methods. You can also add physical characteristics to determine the material and structural properties of a nanoparticle. For more information, see Physical Characterization Category.
•
In vitro Characterizations: Add distinctive characteristics or essential features of the nanoparticle determined through analytical methods. You can also annotate nanoparticles with laboratory results obtained under in vitro conditions. For more information, see In Vitro Characterization Category.
•
Reports: Lists report files or other associated files with composition or characterizations attributes for the selected nanopartical. These files are added using the Reports menu option.
•
In Vivo Characterizations: Currently not available.
If information is filled out for the nanoparticle, you can add to or edit items with the Add or Delete buttons. If you do not have certain security privileges, the Add and Delete buttons will be hidden, and you will have read-only access to the items.
