Particle Tree
The Particle Tree displays nanoparticle annotation information for the selected nanoparticle in the left-hand panel of the browser. Initially, only the particle type and ID you have selected in the Nanoparticle General Information page display. The Particle ID hypertext link on the Particle Tree takes you back to the General Information page that just closed.
If other annotation information has already been filled out for the nanoparticle, it displays in abbreviated form on the Particle Tree.
•
To add to, or edit the annotations, click the add button to the right of the Particle Tree categories.
•
Click the particle name hypertext link to open annotation details.
Note:
If you cannot edit or add information to a particular category based on security restrictions, the add button to the right of the categories is hidden. In this case, you can only access a read-only view of the characterizations which you are authorized to view.
