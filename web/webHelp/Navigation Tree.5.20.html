Displaying a Summary View of Purity Physical Characterizations
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Purity is a Physical Characterization.
If there are multiple Physical Characterizations added under a subcategory in the Navigation Tree, you can display a summary view of all the characterizations in one table. You can then print the table or save the table to a spreadsheet file.
To display all the characterizations in one table, follow these steps:
1.
Click Purity. The Physical Characterization - Morphology page opens listing all the characterizations in one table, one characterization per row.
2.
For information about the the Summary View, see Characterization Summary View.
3.
Once you are done, you can perform the following characterization tasks:

•
Adding a Purity Physical Characterization
•
Copying Characterizations
•
Deleting a Characterization
