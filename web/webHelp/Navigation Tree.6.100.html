Displaying a Summary View of In Vitro Immune Cell Function Characterizations
If there are multiple In Vitro Characterization added to a Blood Contact subcategory under Toxicity in the Navigation Tree, you can display a summary view of all the characterizations in one table. With additional icons, you can print the table or save the table to a spreadsheet file. You can also display and print the individual detail views of the categoryâ€™s characterizations in one table.
