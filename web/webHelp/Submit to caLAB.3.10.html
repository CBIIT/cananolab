Nanoparticle Characterizations
Nanoparticle characterization, describing distinctive characteristics or essential features of the nanoparticle determined through analytical methods, is necessary to record information associated with nanoparticle synthesis and properties.
Nanoparticles can be characterized in caNanoLab 1.2 by physical characteristics and by in vitro characteristics. In vivo characterization is not implemented in v.1.2; it will come in a later version of caNanoLab.
Each characterization category to which you can add annotations displays a hypertext add to its right on the Particle Tree. Click the add link to open a set of tabs that can connect you to forms where you can add the details of laboratory results as corresponding annotation information.
You have the option to delete existing characterizations for a nanoparticle. You can do so by clicking the Delete button in the Particle Tree or from a characterization details page. For more information see Deleting a Characterization.
