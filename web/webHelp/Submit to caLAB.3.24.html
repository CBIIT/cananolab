Morphology
Morphology, referring to the form and structure of a nanoparticle or one of its parts, is one of the physical characteristics of a nanoparticle.
To record morphology data for a particle, follow these steps:
1.
On the Physical Characterization page, click the Morphology tab to open the form for recording nanoparticle morphology data.
2.
Complete the following fields on the form.
The characters [-?-] indicate you can enter text, or you can select from the drop-down list.
Morphology Characteristic
Description
Summary
 
Characterization Source
Enter the source from which the characterization data is derived, such as a vendor or a laboratory (NCL).
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Particle Tree. The asterisk indicates a required field.
Protocol Name - Version
Enter the protocol from which the data is derived and or enter identifying version information for the protocol from which the data is derived.
Description
Enter information not covered by other fields on the form.
Instrument Information
Instrument Type
Identify the kind of instrument used to obtain data.
Instrument Manufacturer
Specify the manufacturer of the instrument used to obtain the data.
Description
Enter information about instrumentation not covered by other fields.

Type

Characterization File/Derived Data Information
Click the Add File/Derived Data link to extend the page where you can add characterization files consisting of graphs, spreadsheet or images, or derived data.
Note:
You can add as many files and derived data as you wish, or you can add derived data without adding files.
File Type
Specify the type of characterization file.
Data Category
These options are dependent upon the characterization type. For example, for Size, options include Intensity Distribution, Number Distribution and Volume Distribution. Select the choice appropriate for your nanoparticle.
File Name
This field remains empty until you select the file to be uploaded. To do so, click the Load File button, opening the Load Characterization File page. For more information about completing the form, see Loading Characterization Files.
File/Derived Data Description
Enter an appropriate description for the file or derived data.
Add Derived Data
Click the hypertext link to open fields for selecting or entering names or values for the derived data. These fields can be specific for the type of characteristic being defined. For example, the Name drop down list is appropriate for Size characterizations: PDI, Peak I, RMS size and Z Average.
Copy to Other Particles
If appropriate, this option copies files and summary information to one or more selected particles from the same vendor. Select one or more particle(s) to which the copied data is to be transferred. (Only appropriate particles are listed.) For more information, see Copying Characterizations
Also Copy Derived Data?
Select this button to copy derived data, as well, to the selected particle.
3.
Once you have entered the appropriate information on the form, click Submit.
OR
4.
Click Reset to clear the fields on the form.
