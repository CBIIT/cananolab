Displaying a Summary View of Size Physical Characterizations
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Size is a Physical Characterization.
If there are multiple Physical Characterizations added under a subcategory in the Navigation Tree, you can display a summary view of all the characterizations in one table. You can then print the table or save the table to a spreadsheet file.
To display all the characterizations in one table, follow these steps:
1.
Click Size. The Physical Characterization - Size page opens listing all the characterizations in one table, one characterization per row.
2.
The following table lists and describes the fields that appear for the characterization.
If the table is missing information or empty, the following fields were not filled in when the characterization was added
Field
Description
View Title
A unique and meaningful name for the characterization. To display a detail view of the size characterization, click the hypertext title name (see Displaying a Detail View of a Size Physical Characterization).
PDI
Protein Disulfide Isomerase (PDI).
Z Average (nm)
The Z average in nanometers.
Characterization File
A link to or a graphic representation of each characterization file which may be a graph, spreadsheet or image, or derived data. The characterization files are numbered in the order they were defined.
To view the characterization file graphic, click the graphic in the table. The graphic opens in a new, larger window.
Instrument Type
The instrument used to obtain data, the abbreviation of the instrument, and the manufacturer of the instrument.
3.
The following table lists and describes the icons and additional tasks that you can perform:
Icon
Description
When you hover over the icon, a popup menu opens.
Print Summary prints the currently displayed summary where all the characterizations are in one table.
Print Full Summary prints each individual characterization in the category in a separate table.
When you hover over the icon, a popup menu opens.
Export Summary exports to one spreadsheet file the currently displayed summary where all the characterizations are in one table.
Export Full Summary exports to one spreadsheet file each individual characterization in the category in a separate table.
4.
