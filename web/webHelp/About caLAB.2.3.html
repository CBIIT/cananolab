caLAB Welcome Login Page
On the caLAB Welcome login page, you can perform the following tasks:
•
Learn about the caLAB application.
•
Update the password for your user account. For more information, see Logging into caLAB.
•
Log in with an existing user account. For more information, see Logging into caLAB.
Before you can login, you must apply for an account to the caLAB Admin who will create the user account for you. You will be notified when the account is ready. caLAB validates your login and alerts you if you have used an incorrect user name or password.
•
Access caLAB Quick Links (hypertext links in the left sidebar of the browser) that include Quick Links to the National Cancer Institute (NCI) website, the NCI Center for Bioinformatics website, and the Nanotechnology Characterization Laboratory (NCL) website.
•
Open the caLAB Online Help by clicking the Help hypertext link on the menu options at the top of the browser.
•
Open caLAB page-sensitive help topics by clicking the Help button at the top of each page or form.
