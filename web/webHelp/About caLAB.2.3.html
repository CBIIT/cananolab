caNanoLab Welcome Login Page
On the caNanoLab Welcome login page, you can perform the following tasks:
•
Learn about the caNanoLab application.
•
Update the password for your user account. For more information, see Logging into caNanoLab/Updating Password.
•
Log in with an existing user account. For more information, see Logging into caNanoLab/Updating Password.
Before you can login, you must apply for an account to the caNanoLab Admin who will create the user account for you. You will be notified when the account is ready. caNanoLab validates your login and alerts you if you have used an incorrect user name or password.
•
Access caNanoLab Quick Links (hypertext links in the left sidebar of the browser) that include Quick Links to the following websites:
•
National Cancer Institute (NCI)
•
NCI Center for Bioinformatics
•
Nanotechnology Characterization Laboratory (NCL)
•
Open the caNanoLab Online Help by clicking the Help hypertext link on the menu options at the top of the browser.
•
Open caNanoLab page-sensitive help topics by clicking the Help button in the login section of the pafe.
