caNanoLab Welcome Main Page
Once logged in on the caNanoLab Welcome Login Page, caNanoLab opens to a second Welcome page. The options available on this page depend upon the permissions granted when your user account was created.
All users can perform these tasks:
•
Access caNanoLab Quick Links (hypertext links in the left sidebar of the browser) that include Quick Links to the National Cancer Institute (NCI) website, the NCI Center for Bioinformatics website, and the Nanotechnology Characterization Laboratory (NCL) website.
•
Use caNanoLab menu options, displayed according to assigned permissions. Available options which may be available are described as follows:
Menu Options
Description
Workflow
caNanoLab provides support for executing assays performed by the NCL. This feature is only available to NCL researchers. Users with permissions can click this option to execute a workflow. For more information, see Executing a caNanoLab Workflow.
Inventory
caNanoLab supports the accessioning of samples and creation of aliquots within the NCL. This feature is only available to NCL researchers. Users with permissions can click this option to work with samples and aliquots. For more information, see Managing caNanoLab Inventory.
Search
caNanoLab allows NCL researchers to search samples, aliquots, workflows, nanoparticles, and nanoparticle reports in the caNanoLab database.
Authorized personnel can search for nanoparticle characterizations and reports. Users with permissions can click this option to initiate searches of these caNanoLab objects. See Launching a caNanoLab Search for more information.
Search Characterizations
caNanoLab allows authorized researchers to search for nanoparticle characterizations and reports.
Note: This menu option displays only for logged in users with other search limitations, based upon assigned permission.
Submit
caNanoLab facilitates the submission of nanoparticle annotations (characterizations, assay results) and reports. This feature is only available to NCL researchers. Users with permissions can select one of these options to annotate nanoparticles or to publish a report. See Submitting Annotations for Nanoparticles or Publishing a Nanoparticle Report.
Administration
caNanoLab allows authorized administrators to manage user accounts, groups, roles, and protection elements.
This feature is only available to NCL users with with this assigned permission.
Home
Click this option to return to the caNanoLab login page. Clicking this button logs you out automatically.
Help
Click to open full help features. For more information see Using caNanoLab Online Help.
Logout
Click to logout of caNanoLab.
The flowchart on the main page illustrated caNanoLab tasks and their functionality. For more information, see caNanoLab Functionality Overview.
