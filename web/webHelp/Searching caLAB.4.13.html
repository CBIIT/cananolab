Nanoparticle Details
The Nanoparticle General Information page opens when you click any of the hypertext links in the Nanoparticle ID column on the Nanoparticle Search Results page.
The page that displays is the same General Information page that displays when you annotate a particle. Details are described in the following table.
General Information Characterization Fields
Definition
Particle Type
The nanoparticle type. Possible types are as follows:
•
Carbon Nanotube
•
Complex Particle
•
Dendrimer
•
Emulsion
•
Fullerene
•
Liposome
•
Metal Particle
•
Polymer
•
Quantum Dot
Particle ID
The particle ID that you selected in the Nanoparticle Search Results page.
Keywords
Keyword(s) the submitter selected to associate with the selected nanoparticle.
Visibility
The group(s) which have access to the annotation information.
