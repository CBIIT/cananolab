Sample Composition
The sidebar options discussed in this topic display only for a nanoparticle already submitted to caNanoLab. For more information, see Understanding the Navigation Tree.
When you click the Navigation Tree Sample Composition link in the left sidebar, the Sample Composition Properties subcategories display:
•
Nanoparticle Entity
•
Functionalizing Entity
•
Chemical Association
•
Composition File
You can click the link again to collapse the subcategories. If annotation information is filled out for the nanoparticle, the annotations are listed under the Composition categories.
You may have the option to add or delete composition details with the Add and Delete buttons. If you do not have certain security privileges, the Add and Delete buttons are hidden, and you have read-only access to the existing items.
See also Adding Sample Composition Annotations.
