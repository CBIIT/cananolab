Sample Details Page
The Sample Details page opens when you click the View Details button in the Sample Search Results page. The page displays comprehensive information about the corresponding sample hit, both about the sample (first table) and the container(s) in which it is stored (second table). Details were defined when the samples were created in caNanoLab; they are described in the following tables. Columns are sortable by clicking the column header.
Sample Fields
Description
Sample ID
The fully qualified name of the nanoparticle to be searched.
Type
The name of the sample type being searched. Example: Carbon Nanotube.
Description
Description of the sample
Source
The originating source for this sample
Source ID
The unique identifier for the sample
Date Received
The date the sample was received by the lab
Solubility
Descriptive solubility information for the sample
Lot ID
Unique ID for the sample lot
Lot Description
Description of the sample lot
Number of Containers
Number of containers in which the sample is stored
General Comments
Comments about the sample not covered by the other fields in the table.
Container {ID} Fields
Description
Container Type*
The type of container in which the sample is stored
Initial Quantity
The weight of the sample that was stored in this container initially.
Concentration
The value of the Concentration of the sample in the current container.
Note: If Concentration is described, Diluents/Solvents are also described.
Volume
The volume in each container of sample.
Diluents/Solvent
Diluents: The liquid used to dilute the liquid sample.
Solvent: The solvent used dissolve the solid sample.
 
Note: If Diluents/Solvents is entered, Concentration is also described.
Safety Precautions
Safety precautions associated with the sample, e.g. toxicity.
Storage Conditions
The temperature and other variables used to store the sample in the container.
Storage Location
Enter or select the location (Room, Freezer, Shelf, Box) in which this container is stored. Example: Freezer1, Box 12.
Comments
Additional comments that are associated with this particular sample container.
