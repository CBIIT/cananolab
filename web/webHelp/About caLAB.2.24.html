Working with Files
Within a workflow, when you click consecutively on (+) symbols to open an assay and an associated run, if the In or Out folder for the run has a (+) symbol, it is populated with either aliquots or files or both. Click the In or Out text itself. At that point, several hypertext links become available in the right-hand pane. Some of these links govern work with files in caLAB.
By using these links, you can:
•
Upload input and output files
•
Download any files you have uploaded
•
Mask files
