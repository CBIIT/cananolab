Submitting a Nanoparticle Annotation
Annotating a nanoparticle involves selecting the nanoparticle to annotate, entering general information about the nanoparticle, submitting the particle function, and entering the particle characterizations (physical, in vitro). Note that annotating a nanoparticle with characterizations includes submitting the source of the characterization, the instrument used to perform the characterization, where appropriate, an appropriate protocol, the characterization information itself, and relevant information in the form of uploaded files. A single particle can have multiple annotations, either coming from different sources, or the same data source being tested under different conditions (e.g. temperature, solvents, instruments).
Only users with PI permissions can annotate a nanoparticile. For more information, see Permissions and Visibility.
To annotate a nanoparticle, select Submit > Nanoparticle, which opens an Annotate Nanoparticle summary page. This page describes a nanoparticle and offers the following two links for working with nanoparticles.
Annotate a New Nanoparticle
This link opens a form for annotating a nanoparticle in the caNanoLab database to which no annotations have yet been added. For more information, see Nanoparticle General Information.
Select an Existing Nanoparticle
This opens the form for launching a search for an existing nanoparticle to which annotations have already been added. For more information, see Searching Nanoparticles.
