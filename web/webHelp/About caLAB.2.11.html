Creating an Assay Run
Important!
Before an Assay Run can be created in caLAB, it is necessary to create one or more samples, then create aliquots of those samples. For more information, see Managing caLAB Inventory.
Notes:
In caLAB 1.0, you can create Runs from two vantage points:
•
Create Assay Run--If you launch its creation from inside a workflow assay, it is called an “Assay Run” because you are creating a Run for a selected Assay node on the tree.
•
Create Run--If you launch its creation from the Create Run dialog box that opens as you “execute a workflow”, it is called simply a “Run”. As you create it, you must specify the assay with which it is to be associated.
Before a Run can be created in caLAB, it is necessary to create one or more samples, then create aliquots of those samples. Aliquots are essential components of a Run. For more information, see Creating a New Sample and/or Creating A New Aliquot.
To create a Run already assigned to an existing assay, follow these steps:
1.
Select Workflow > Execute Workflow. The workflow in the caLAB system displays in the left panel of the browser. Each item in the workflow for which information currently exists is preceded by a (+) and a “folder” icon. (Items with no current information display a “piece of paper” icon.
2.
Click on the (+) in front of Prescreening or In Vitro on the workflow tree. Click on the (+) in front of the PCC, STE, GTA or ITA listings. Note that there are pre­viously submitted Runs for each, numbered consecutively.
3.
Click on any of the PCC, STE, GTA or ITA hypertext links. This opens a Create Run for {Assay name} dialog box in the right pane of the browser.
4.
Continue with step #2 below.
OR to create a run from outside the workflow tree, follow these steps:
1.
Select Workflow > Execute Workflow. This opens a Create Run dialog box in the right pane of the browser.
2.
In the Create Run ... dialog box that opens, select or enter information in the text fields and drop-down lists. Items with an asterisk are required fields.
Assay Run Text Fields
Description
Assay Type:Assay (This option is available only if the assay to which the run is to be assigned has not yet been determined.)
Select in the drop-down list the assay to which the run is to be assigned. All assays previously submitted to caLAB are on the list.
Aliquots
--The Aliquots select list displays all available aliquots in the system.
--The Use Aliquots panel displays aliquots that are assigned for this Run.
 
Select one or more aliquots in the Aliquots list using conventional selection techniques, and click the Forward button (>>) to assign the selected items to the Use Aliquots panel.
Aliquot Comments
Enter comments pertinent to the aliquot selections, for example, you may describe important observations regarding their use.
Run By*
Select in the drop-down menu the name of the person who physically ran this experiment in the lab. Anyone who has caLAB user account displays in this list.
Run Date*
Click the calendar to locate the date on which the experiment was run.
3.
Click the Submit button to enter the Assay Run.
As soon as you submit the run, the Assay Run is included in the list under the Assay you selected, with a folder icon, and Upload Input File or Upload Output File hypertext links display in the righthand panel. Click one of these options to upload any files from your local system that are pertinent to the Assay Run you have created. For more information, see Uploading Input/Output Files.
