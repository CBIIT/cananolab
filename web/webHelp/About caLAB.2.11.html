Managing caNanoLab Inventory
When you click the Inventory option on the menu bar, tasks associated with inventory management become available:
•
Managing Samples
•
Manage Aliquots
Each of these “Manage” options allows you to “select an existing sample (or aliquot)”. If you select either of those hyperlinks, a search form opens where you can initiate a search for an existing sample or aliquot. Alternatively, you can go directly to the search function by clicking one of the “Searching...” options under Inventory. For more information, see Launching a caNanoLab Search.
