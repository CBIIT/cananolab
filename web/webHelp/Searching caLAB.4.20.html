Remote Searches
The Remote Nanoparticle Search feature searches for publicly available nanoprticles from all active caNanoLab data services connected to the caBIG™ grid (caGrid™). The Remote Report Search searches for publicly available nanoparticle reports from all active caNanoLab data services connected to the caGrid. For more information about caGrid, see https://cabig.nci.nih.gov/workspaces/Architecture/caGrid.
An added benefit is that, not only are searches conducted on caGrid, but this feature publishes the caNanoLab domain model on caGrid for the benefit of other users of caNanoLab.
The Remote Searches menu tab in caNanoLab offers two features, Remote Search Nanoparticles and Remote Search Reports. In each one of the search pages, a list of the active grid node host names, which are discovered at run time, display. You can choose one or more of them to search. Grid node host name(s) also display in the result.
