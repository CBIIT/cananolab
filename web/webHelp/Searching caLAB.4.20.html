Remote Reports Search
The Remote Reports Search allows you to search remote caNanoLab nodes connected to the caGrid for available nanoparticle reports. Only publicly available reports are retrieved in this search.
To launch this search, follow these steps:
1.
Select Remote Searches > Remote Search Nanoparticles.
2.
In the dialog box that opens, enter text or select options as appropriate (described in the following table). Add an asterisk for a wildcard search.
If you click the Search button having defined no parameters, caNanoLab searches all remote caNanoLab nodes for available nanoparticle reports.
