Workflow Overview
Workflows in caLAB must be incorporated into the system by the caLAB team; they become workflow templates. caLAB 1.0 has implemented one workflow, a reference workflow contributed by the Nanotechnology Characterization Laboratory (NCL). NCL researchers can use the workflow template to load assay input files, assign aliquots, and load output files (assay results)
Run Tree
Once you select to execute a workflow, caLAB allows you to create a new assay run or to search for an existing run. When you create a new Assay Run, caLAB displays the selected Run (Run Tree) in the left hand panel. This facilitates easy navigation within the Run. When you search Runs, described in Searching Runs, search results display in table format. From the table, you can choose a Run to view in detail, while a Run Tree displays in the left side panel.
Upon completing either of these two tasks under Execute Workflow (Executing a caLAB Workflow), the Assay Run displays in the Run Tree in the left-hand pane of the caLAB browser. The workflow displays pre-populated data. There are four assay types in this workflow--Prescreening, Physical, In Vitro, and In Vivo and PCC. Each of the items listed under the assay types are assays. Only two assay types, Prescreening and In Vitro, are populated with data in caLAB 1.0.
