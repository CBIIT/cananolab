Searching Existing Aliquots
This function is limited to users belonging to a specific research group. For more information, see Permissions and Visibility.
To search for a specific aliquot, follow these steps:
1.
Click the Samples caNanoLab menu option. On the Manage Sample/Aliquot Home page, click the Search Existing Aliquots link. This opens the Search Aliquots form.
2.
Enter or select the Search information described in the following table. In any text field, you can add an asterisk for a wild-card search. For example, in searching for aliquot NCL-21-1-4, you could enter NCL-21*.
There are no required fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab aliquots will be listed in the Search Results.
Name
Description
Aliquot ID
Enter the fully qualified name of the Aliquot to be searched. Add an asterisk for a wild-card search.
Sample Type
Select from the drop-down list the name of the Sample Type being searched. Example: Emulsion, Liposome.
Source
Select from the drop-down list the name of the Source from which the aliquot originated. Example: XYZ institute.
Source ID
Select from the drop-down list the unique identifier for each aliquot source.
Date Accessioned
Click the two calendars to define the range of dates during which the all aliquots were accessioned in the lab.
Creator
Select from the drop down list the name of the person who created the aliquot.
Storage Location
Enter or select the location (Room, Freezer, Shelf, Box) in which this container is stored. Example: Freezer1, Box 12.
To clear all fields and start again, click the Reset button.
3.
Click Search to execute the search.
