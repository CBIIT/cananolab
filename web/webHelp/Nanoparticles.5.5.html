Submitting a Nanoparticle - General Information
If you perform a nanoparticle search, on the Nanoparticle Search Results page, click the hypertext Particle ID in the first column. This Nanoparticle General Information page opens, allowing you to review the general information about the particle that was defined when it was submitted. See Nanoparticle Details for more information.
To enter general information about the nanoparticle, follow these steps:
1.
Click the Nanoparticles caNanoLab menu option. On the Manage Nanoparticle Home page, click the Submit a New Nanoparticle link. This opens the Submit Nanoparticle-General Information form.
2.
Select or enter the appropriate information in the text fields and drop-down list. Fields are described in the following table.
Items with an asterisk are required fields.
The characters [-?-] indicate you can enter text, or you can select from the drop-down list
Name
Definition
Particle Type *
Select the nanoparticle type from the drop-down list. For information about each type, see Composition Category.
For whatever particle type you select, corresponding particle IDs become available in the Particle ID drop-down list.
Note:
Only particle types that have un-annotated sam­ples display in the drop-down list.
Particle ID*
Every sample submitted to caNanoLab is identified by a nanoparticle type. When you identify the particle type in the first field of this dialog box, caNanoLab lists all particle IDs in the Particle ID field that correspond to that particle type.
Select the Particle ID for the nanoparticle of that particle type to which you want to add annotations.
Keywords
Add any keyword(s) you want associated with the selected nanoparticle, one keyword per line. You can enter something not necessarily captured by any data. The keywords you enter can be used to search for this particle ID.
Visibility
Visibility, set by the person who enters the annotation information, determines access to the annotations for the selected particle ID. (Groups listed are user groups which have been defined in the caNanoLab security model.) Select one or more groups you wish to have access to the annotations you are defining.
Note: This option sets visibility requirements for the nanoparticle characterization. You can set a different visibility when you submit assay results files, restricting access to the actual files. For more information, see Loading Physical Characterization Files.
To clear the entries and start again, click the Reset button.
3.
When you have finished defining the nanoparticle, click the Submit button. A message appears indicating the submission.
The Navigation Tree opens in the left hand panel of the browser. For more information, Understanding the Navigation Tree.
