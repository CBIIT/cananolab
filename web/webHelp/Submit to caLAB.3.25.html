Surface
Surface, referring to the extended two-dimensional outer boundary of a nanoparticle, is one of the physical characteristics of a nanoparticle.
To record surface data for a particle, follow these steps:
1.
On the Physical Characterization page, click the Surface tab to open the form for recording nanoparticle surface data.
2.
Complete the following fields on the form:
Surface Characteristic
Description
Characterization Source*
Enter the source from which the characterization data is derived, such as a vendor or a laboratory (NCL).
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Particle Tree. The asterisk indicates a required field.
Protocol Name
Enter the protocol from which the data is derived.
Protocol Version
Enter identifying version information for the protocol version from which the data is derived.
Description
Enter characterization information not covered by other fields on the form.
Instrument Information
Instrument Type
Identify the kind of instrument used to obtain data.
Instrument Type Abbreviation
This is a non editable field. Abbreviation of the selected instrument type will be displayed here if the abbreviation exists.
Instrument Manufacturer
Select the manufacturer of the instrument used to obtain the data.
Description
Enter information about instrumentation not covered by other fields.
Particle Surface Information
 
Surface Area
Enter the measurement of the surface area; the default unit is Square nm.
isHydrophobic
Select Yes or No, indicating hydrophobicity, the property meaning that the particle does not dissolve in, absorb, or mix easily with water.
Charge
The charge is the quantity of unbalanced electricity in a body (either positive or negative) and construed as an excess or deficiency of electrons.
Enter the information about the charge on the particle surface; select from the drop-down list the correct unit correctly describing your entry.
Zeta Potential
Zeta potential is the magnitude of the electrical potential as generated by ion accumulation at the particulate surface. This surface consists of two layers, the stern layer and the diffuse layer, which comprise an electrical double-layer. The measurement of the electrical potential. provides important information regarding the dispersion mechanism, the degree of particle dispersion and the electrostatic potential of the sample.
Enter the measurement in the text box.
Surface Chemistry
 
Number of Surface Chemistry
Enter the number of Surface Chemistry Distributions.
Update Surface Chemistries
Note: If you enter a number greater than “0”, complete all of the additional information for the Sample, then click the Update Surface Chemistries button. This extends the page so you can enter information for each unit. See below.
Molecule
Enter the molecule making up the surface chemistry of the nanoparticle.
Number of Molecule
Enter the number of molecules on the surface area.
Characterization File Information
 
Number of Files
Enter the number of Files.
Update Files
Note: If you enter a number greater than “0”, complete all of the additional information for the Sample, then click the Update Files button at the bottom of the page. This extends the page with the corresponding number of additional fields so you can enter information for each unit.
Characterization File Name
This field remains empty until you select the file to be uploaded. To do so, click the Load File button, opening the Load Characterization File page. For more information about completing the form, see Loading Characterization Files.
