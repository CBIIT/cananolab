Executing a caLAB Workflow
To execute a workflow, select Workflow > Execute Workflow. When you select this menu option, two hypertext links display that allow you to perform the following tasks:
•
Create a New Assay Run
•
Select an Existing Assay Run
Click either of these links to continue following your selected workflow.
Working in a workflow consists of creating Assay Runs with associated aliquots and input and output files. You can also add aliquots and files to existing runs.
