Creating a New Sample
In a laboratory setting, when a new sample comes in from an outside source, you would log it in and store it in a repository. For using it, you would dissolve or dilute the sample, then use it to create aliquots for use at the bench following customized protocols. For more information, see caNanoLab Sample Overview.
In caNanoLab, if you have NCL researcher privileges, you can “create” a sample, by recording a physical description of the source sample and its origin. Then you “create” associated aliquots by describing the process by which the source sample is dissolved and/or diluted and otherwise prepared for bench use. You can also view associated protocols that have been previously implemented into caNanoLab.
Notes:
Protocols must be submitted separately to the caNanoLab team for incorporation into the caNanoLab system.
Before an Assay Run can be created in caNanoLab, it is necessary to create one or more samples, then create aliquots of those samples.
To create a Sample, follow these steps:
1.
Click Inventory > Manage Samples. This displays two links; click Create a New Sample. This opens the Create Sample form.
2.
Enter or select the appropriate information about the sample you are creating in the text entry fields (described in the following table). Lists associated with drop-down menus are predefined in caNanoLab. Items that have an asterisk are required fields.
Create Sample Text Fields
Description
Sample ID Prefix*
Represents the fully qualified name of the Sample. This field is pre-determined by the system as the sample is created, however, the field can also be edited. Example: NCL-8. (NCL is the prefix for the NCL lab.) For more information, see Aliquot Naming Conventions.
Sample Type
Select from the drop-down menu the name of the Sample Type to which the current sample belongs. Example: Dendrimer, Liposome.
Description
Enter additional comments relevant to the sample creation process.
Source
Enter the name of the originating source for this sample.
Source ID
Enter the unique ID that was assigned by the organization where the sample originated.
Date Received
Click the calendar to locate the date on which the sample was received by the lab from the source.
Solubility
Enter the value representing how much of a given substance dissolves in a particular solvent.
Lot ID*
Enter the identifier of the lot such as the lot number for this Sample. This is required for NCL, but “N/A” also permissible.
Lot Description
Enter a description about the lot to which this Sample belongs.
SOP
Select the Standard Operating Procedure (SOP) type that was used by the lab to generate the sample. You can leave the text field blank, or select between Original or Testing.
Note: SOPs can be submitted to caNanoLab for inclusion in the software as Protocols.
Number of Containers* (Update Containers button)
Enter the number of containers in which the Sample is stored. The container information would dynamically grow based on the Container Number field. (For NCL, “1” is the only permissible value in caNanoLab 1.1.)
Note: If you enter a number greater than “0”, complete all of the additional information for the Sample, then click the Update Containers button at the bottom of the page. That way, the information you enter becomes part of your container template and you don’t have to re-enter it again. This is particularly helpful when you are submitting a large number of sample containers. See Step #3.
General Comments
Enter additional information about the sample creation not covered by other fields in the form.
Container (Template Container)
Container Type*
Select the Container Type in which the sample is stored in the pre-populated drop-down list. Options: Tube, Vial or Other.
Quantity
In the first text box, enter the weight of the sample that is stored in this container. In the drop down list, select the unit used.
Concentration
In the first text box, enter the value of the Concentration of the sample in the current container. In the drop down list, select its associated measurement unit.
Note: If Concentration is entered, Diluents/Solvents must also be entered and vice versa.
Volume
In the first text box, enter the value of the Volume in each container in the first text box. In the drop down list, select its associated measurement unit.
Diluents/Solvent
Diluents: Name the liquid used to dilute the liquid sample.
Solvent: Name the solvent used dissolve the solid sample.
Note: If Diluents/Solvents are entered, Concentration must also be entered and vice versa.
Safety Precautions
Enter safety precautions that are worth noting.
Example: The sample is Toxic.
Storage Conditions
Enter the temperature and other variables required to adequately store the sample in the container.
Storage Location
Enter in one or more fields the location in which this container is stored. Example: Freezer1
Comments
Enter additional comments that are associated with this particular sample container.
3.
After you click the Update Containers button, the number of containers you entered is reflected by the number of forms that now display in the browser. Modify each form appropriately to customize the description for each container.)
4.
Click the Submit button after defining the sample(s) appropriately.
After you have clicked the Submit button, if you have failed to enter appropriate information for particular fields, such as the solvent when concentration is entered or the container volume unit, you will receive an error message. You must correct the error before your submission is accepted.
The detailed information you entered displays in table format. A separate table dis­plays for each container you described in your entry.
Samples are used to create aliquots that become part of Assay Runs. For more information see Creating A New Aliquot.
