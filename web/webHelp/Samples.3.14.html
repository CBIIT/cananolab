Aliquot Search Results
If you click the Search button having defined no parameters, all of the caNanoLab aliquots will display in the search results.
Aliquot Search Results display in the browser window in table format. Column headings are appropriate for the aliquot for which you searched. Columns are sortable by clicking the column header.
Name
Description
Select
Select one or more of the search hits for which you want to view details. Click the View Details button.
Sample ID
The fully qualified name of the source sample for the aliquot.
Sample Accession Date
The date the source sample was accessioned in the laboratory
Sample Submitter
The name of the person who submitted the sample..
Aliquot ID
The ID of the aliquot located in the search
Aliquoted Date
The date the aliquot was created.
Aliquot Location
The location where the aliquot is stored.
Aliquot Creator
The name of the person who created the aliquot.
For any hit for which you want more information, select the corresponding radio button. Click the View Details button to open details for a selected aliquot on the Aliquot Details Page.
