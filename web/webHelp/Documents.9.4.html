Submitting a Report
This help topic opens from the Update Report page, as well as from the Submit Report page. For more information about the Update Report page, see Update Report Page.
The Submit Report feature allows you to upload a report and associate it with one or more nanoparticles.
To upload a nanoparticle report, follow these steps:
1.
Click the Documents caNanoLab menu option. On the Manage Documents home page, click the Submit a New Report link. This opens the Submit Report form.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Asterisks indicate required fields.
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear
Nanoparticle Report Criteria
Description
Upload Report File
or
Enter Report URL
Click the Browse button to navigate to the file to be associated with the report, or enter the URL where the report is located.
Submitted Report
This category displays only if you are viewing/editing an existing report. Click the hypertext report name to open the report.
Report File Title*
Enter the report title.
Nanoparticle Sample Name
Select one or more nanoparticle sample names, to be the subject of the report, from the list.
Report File Description
Enter information not covered by other fields on the form.
Visibility
The term, Visibility, is used to describe the access to objects granted to logged in users. Select the group to which this report is to be visible.
3.
To clear the fields and start again, click Reset.
4.
Once you have entered the appropriate information on the form, click Submit.
