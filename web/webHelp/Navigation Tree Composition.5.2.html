Understanding the Navigation Tree
Once you submit or search and select an existing nanoparticle with the Nanoparticles menu option (see Nanoparticles Overview), the Navigation Tree heading displays in the left-hand panel of caNanoLab with the selected “Viewing Particle [name]”. The Navigation Tree comprises categories which you can use to add annotations to the nanoparticle. Each category is a button, and when you click a button, options may expand below to show subcategories. Click the heading again, and the subcategories collapse.
The following list provides a general description of how you can use each category.
•
General Information: Display general and basic information about the nanoparticle entered during the submission process.
•
Sample Management: Option currently not available.
•
Sample Composition: Add composition properties to the nanoparticle. For more information, see Sample Composition.
•
Physical Characterizations: Add distinctive characteristics or essential features of the nanoparticle determined through analytical methods. Add physical characteristics to identify the material and structural properties of a nanoparticle. For more information, see Navigation Tree—Physical Characterizations.
•
In Vitro Characterizations: Add distinctive characteristics or essential features of the nanoparticle determined through analytical methods under in vitro conditions. For more information, see Navigation Tree—In Vitro Characterizations.
•
Documents: List report files or publications with composition or characterizations attributes for the selected nanoparticle. These files are added using the Documents menu option. See Adding Documents to a Nanoparticle.
Once a document has been submitted and associated with a specific nanoparticle, when the nanoparticle is opened in caNanoLab, the associated documents are listed along with the particle characterizations in the left sidebar. Where appropriate, the document list includes hypertext links so you can readily open the document(s).
•
In Vivo Characterizations: Currently not available.
