Report Page
After you have performed a document search (Searching Nanoparticle Documents) the Nanoparticle Documents Search Results page opens. Click the hypertext Title in the first column for a report to open the Report page.
On this page, you can review and/or edit the information that was defined when the report was submitted. The page lists three pieces of information about the report:
1.
Report title
2.
A hypertext link to the report itself, which can be in several different formats such as PowerPoint, TXT or PDF. To open the report, click this link.
3.
Description of the report
Click the following icons to perform report-related tasks:
Edit the report.
Print report details (from this Report page).
Export report details (from this Report page) to MS Excel.
