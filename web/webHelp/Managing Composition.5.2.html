Understanding the Navigation Tree
Once you submit or search and select an existing sample with the Samples menu option (see Samples Overview), the Navigation Tree heading displays in the left-hand panel of caNanoLab with the selected [Sample name]. The Navigation Tree comprises categories which you can use to add annotations to the sample. Each category is a button. When you click a button, tabs appear in the right hand panel to show corresponding subcategories. With appropriate permissions, on the tabs you can add annotations or publications relevant to the sample. The tabs also display annotations or publications that were already added to the open sample.
The following list provides a general description of how you can use each Navigation Tree category.
•
Sample Information: This displays after you click the sample name itself. It shows general and basic information about the sample entered during the submission process.
•
Composition: Composition annotations define nanomaterial and functionalizing entities such as nanoparticle type and components like antibodies that performs sample functions. For more information, see [Sample] Composition Overview.
•
Characterizations: Characterization annotations define essential characteristics determined through analytical methods: physico-chemical features relating to molecular weight or size, for example, or under in vitro conditions such as blood contact or toxicity, or in vivo conditions such as pharmacokinetics. These characterizations add physical characteristics to identify the material and structural properties of a sample. For more information, see Managing Sample Characterizations.
•
Publication: Publication tabs display publications such as articles, books chapters, reviews and reports that have already been added or can be added to a sample. You can also add publications using the Publications menu option. See Adding Publications to a Sample.
