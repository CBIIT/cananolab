Displaying a Detail View of an Cytokine Induction In Vitro Characterization
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations.
Cytokine Induction is an In Vitro Characterization.
If there is a Cytokine Induction In Vitro Characterization added to the Navigation Tree, you can display a detail view of the characterization in a table. You can then print the table or save the table to a spreadsheet file.
To display a detail view of an existing plate aggregation characterization, follow these steps:
1.
Click the hyperlinked name in the Navigation Tree. The In Vitro Characterization - Cytokine Induction page opens.
2.
For information about the the Detail View, see Characterization Detail View.
3.
Once you are done, you can perform the following characterization tasks:

•
Adding a Cytokine Induction In Vitro Characterization
