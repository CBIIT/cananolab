Update Report Page
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle report results from the Login page.
The [Document] Update page opens when you click any of the hypertext file name links on the Report Page or the Publication Page.
The page that displays is the same Report page on which the document was submitted into caNanoLab. Details are described in the following table. On this page, you can review or edit report information.

The term, Visibility, is used to describe the access to objects granted to logged in users. Select the group to which this report is to be visible.
