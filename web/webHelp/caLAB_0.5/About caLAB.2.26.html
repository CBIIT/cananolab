Workflow Search Results
Workflow search results display in the viewer window, summarizing the Assay Run information in table format. Column headings are the same as the text fields you defined in the search criteria. Displayed file names are hypertext links which you can click to view or download the file.
For more information about configuring a search, see Searching a Workflow.
