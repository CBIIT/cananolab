Masking a File
Once you have uploaded a file, it is listed in the In folder of a run, along with other input files you may have uploaded. In caLAB 0.5, you cannot delete uploaded files from the system, but you can mask them. As an example for doing so, perhaps you accidentally uploaded the wrong file for an assay, and you do not want it to be recognized as an active file in the assay.
To mask a file, follow these steps:
1.
Click on the (+) symbol to open the In folder. Each item listed in the folder has been assigned to the open Run. Each is a hyperlink that can open its corre­sponding details.
2.
Click the right-most link in the right pane of the caLAB viewer, Mask Files.
The viewer displays in table format general information for the workflow, as well as it lists the uploaded files.
3.
In the Uploaded Input Files table, Mask Action column, click the Mask hyper­text link corresponding to the file to be masked.
Upon masking the file, it is still visible in the workflow tree, but its name has been changed to itacilized font, indicating its masked status. You can still view the file details by clicking its name. The operating system loads the appropriate application for viewing the file.
