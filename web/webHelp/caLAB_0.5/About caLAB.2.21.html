caLAB File Naming Conventions
In the caLAB workflow tree, an uploaded file is identified by its original file name. When you upload a file into caLAB, it is also assigned a caLAB-generated name, using this convention: yearmonthday_hour_min_second_{original file name}. This name does not display in the workflow tree, but if you mouse over the file name, the full name displays as popup text. The full name also displays on the list of files that opens when you click the Mask Files hypertext link when a Run is selected.
The time stamp, accurate to a millisecond, is added to a file name during the upload procedure. You can upload duplicate files (as in a case where a file is modified), and the timestamp would differentiate between the two. The caLAB system can identify the person who uploaded a specific file.
