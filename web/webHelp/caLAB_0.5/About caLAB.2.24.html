Searching a Workflow
The Search Workflow feature in caLAB searches a workflow for Run information such as a file, an aliquot, their status, etc. based on query input.
To launch a search for one or more specific components in an assay, follow these steps:
1.
Select Search > Search Workflow. This opens the Search Workflow page.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, in searching for sample NCL-8-234, you could enter NCL-8*.
When defining search criteria, there are no "required" fields. You can specify only those parameters that you consider essential to the search. If you click the Search button having defined no parameters, components of all of the caLAB runs will dis­play in the Search Results.
Workflow Search Text Fields
Description
Assay Name
Enter the name of the assay that is used in the work­flow being searched. Add an asterisk for a wild-card search.
Assay Type
Select from the drop-down list the type of experiment used. Options are preconfigured for the workflow being searched.
Assay Run Date
Click the two calendars to define the range of dates for the search target.
Aliquot ID
Enter the unique ID of the Aliquot that is associated to the Workflow that need to be searched.
File Name
Enter the name of the file(s) that are used as input or result (output).
File Submission Date
Click the two calendars to define the range of dates during which files were uploaded.
File Submitter
Select in the drop-down menu the name of the person who uploaded the file(s).
Join Search Criteria By
Select AND or OR logic before performing the Work­flow Search.
 
Note: With the AND criterion selected, a search that satisfies all the entered and checked criteria is per­formed. With OR selected, a search that satisfies any one of the criteria is performed.
Additional Filtering of Target Data
 
Input Files?
Check the box if you want only input files.
Output Files?
Check the box if you want only output files.
Include Masked Aliquots?
Check the box to search the workflow for aliquots cur­rently masked in the system. Leave the box clear to locate only “active” aliquots.
Include Masked Files ?
Check the box to search the workflow for files currently masked in the system. Leave the box clear to locate only “active” files.
3.
Click Submit to execute the search. To clear all fields, click the Reset button.
For information about search results, see Workflow Search Results.
