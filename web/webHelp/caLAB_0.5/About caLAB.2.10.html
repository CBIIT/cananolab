Creating Aliquots
Before an Assay Run can be performed in caLAB, it is necessary to create one or more samples, then create aliquots of those samples, following the same workflow tasks as they would be performed in a laboratory. (Aliquots are normally created in batches, not singly.)
 
Note: Before you can create aliquots, you must first create a sample, the source from which aliquots are created. For more information see Creating a Sample.
1.
To create an aliquot, click Administration > Manage Aliquots. This opens the Create Aliquot form.
2.
Enter or select the appropriate information about the sample you are creating in the text entry fields and drop-down menus (described in the following table). Lists associated with drop-down menus are predefined in caLAB. Items that have an asterisk are required fields.:
Sample ID* or Aliquot ID*
Select from the drop-down list the name of the sample or aliquot from which the new Aliquot is being created. This is a sample or aliquot that must have previously been created in caLAB. (The IDs are automatically generated in caLAB. For more information see Aliquot Naming Conventions.)
 
Note: If both Sample ID and Aliquot ID are selected from the drop down menu, the system will use Aliquot ID and ignore Sample ID.
Number of Aliquots (Update Ali­quots button)
Enter the number of Aliquots that were created in this batch of Aliquot creation.
 
Note: If you enter a number great than “1”, complete all of the additional information for the Aliquot, then return to click the Update Aliquots button. That way, the information you enter becomes part of your container template and you don’t have to re-enter it again. This is particularly helpful when you are inserting a large num­ber of aliquot containers. See Step #3.
Template Aliquot
Container Type*
Select the Container Type in which the aliquot is stored from the pre-populated drop-down list. Options: Tube, Vial or Other.
Other
If Other [Container Type] is selected, enter the name of the Container Type that is used to store the Aliquot.
Quantity
In the first text box, enter the weight of the aliquot that is stored in this container. In the drop down list, select the unit used.
Concentration
In the first text box, enter the value of the Concentration of the aliquot in the current container. In the drop down list, select its associated measurement unit.
 
Note: If Concentration is entered, Diluents/Solvents must also be entered and vice versa.
Volume
In the first text box, enter the value of the Volume in each container in the first text box. In the drop down list, select its associated measurement unit.
Diluents/Solvent
Diluents: Name the liquid used to dilute the liquid sam­ple.
Solvent: Name the solvent used dissolve the solid sam­ple.
 
Note: If Diluents/Solvents is entered, Concentration­must also be entered and vice versa.
How Created
Select one of the relevant options for the Aliquot you are creating. In caLAB 0.5, the options for creation method are:
•
Solubilized
•
Lyophilized
View Protocol
Click the button to open protocol information describing the laboratory procedure for creating the aliquot. View Protocol opens the document based on the selection in the How Created drop down menu.
 
Note: An available protocol must be submitted exter­nally to the caLAB team for it to be incorporated into and available in caLAB. In caLAB 0.5, available proto­cols have been provided by the NCL.
Storage Conditions
Enter the temperature and other variables required to adequately store the aliquot in the container.
Storage Location
Enter or select the location in which this container is stored. Example: Freezer1
General Comments
Enter additional comments that are associated with this particular aliquot container.
3.
After clicking the Update Aliquots button, you can edit the aliquots you have created. See Editing an Aliquot for more information. Once entered into the database, an aliquot can no longer be edited.
4.
Click the Submit button after defining the sample(s) appropriately.
After you have clicked the Submit button, if you have failed to enter appropriate information for particular fields, such as the solvent when concentration is entered or the container volume unit, you will receive an error message.You must correct the error before your submission is accepted.
The detailed information you entered does not display, but is entered into the caLAB database.
