Working in caLAB
caLAB Workflow Overview
Workflows in caLAB must be incorporated into the system by the caLAB team; they become workflow templates. caLAB 0.5 has implemented one workflow: a reference workflow contributed by the Nanotechnology Characterization Laboratory (NCL). While the workflow is preconfigured and prepopulated with some data, you can add additional data and files to the workflow template.
After you have logged into caLAB, the application opens automatically to a Welcome page that displays a recommended laboratory workflow. The recommended basic workflow is represented in the following figure.
The caLAB online help table of contents also follows the same basic workflow.
A workflow can include any or all of the following tasks:
•
Create sample
•
Create aliquot
•
Execute a workflow
•
Create assay run/run
•
Upload input and output files for runs
•
Mask a file (input and output files)
•
Mask an aliquot
•
Search a workflow for run information; view search results
•
Search for samples and aliquots; view search results
For more information about each task in a workflow, click the appropriate hypertext link.
In this release, one workflow template is implemented in the system: it is a reference inplementation for NCL. If you select Workflow > Execute Workflow, the workflow tree opens in the left-hand pane of the caLAB viewer. The workflow displays pre-populated data. There are four assay types in this workflow--Prescreening, In Vitro, In Vivo and PCC. Each of the items listed under the assay types are assays. Only two assay types are populated in caLAB 0.5.
The assays in the caLAB 0.5 workflow template are described as follows:
