caLAB Welcome Login Page
On the caLAB Welcome login page, you can:
•
Learn about the caLAB application.
•
Register for a new user account
•
Log in with an existing user account.
Before you can login, you must apply for an account to the caLAB Admin who will create the user account for you. You will receive an email with the appropriate login information. caLAB validates your login and alerts you if you have used an incorrect user name or password.
•
Access caLAB Quick Links (hypertext links in the left sidebar of the viewer) that include Quick Links to the National Cancer Institute (NCI) website and the NCI Center for Bioinformatics (NCICB) website.
•
Open the caLAB Online Help by clicking the Help hypertext link on the menu options at the top of the viewer.
•
Open caLAB page-sensitive help topics by clicking the Help button at the top of each page or form.
