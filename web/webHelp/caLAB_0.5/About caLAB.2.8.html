Executing a caLAB Workflow
To execute a workflow, select Workflow > Execute Workflow. When you select this menu option, a workflow tree appears in the left pane of the viewer window. In the rightÂ­hand pane, the Create Run dialog box opens.
Working in a workflow consists of adding Assay Runs with associated aliquots and input and output files. You can also add aliquots and files to existing runs.
