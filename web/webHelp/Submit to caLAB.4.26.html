Nanoparticle Purity
Purity, in the context of caLAB, refers to the degree of being free of contaminants or heterogeneous components. Purity is one of the physical characteristics of a nanoparticle.
To record purity data for a particle, follow these steps:
1.
Click the Purity tab to open the form for recording nanoparticle purity data.
2.
Complete the following fields on the form:
Purity Characteristic
Description
Characterization Source
Enter the source from which the characterization data comes, such as a vendor or a laboratory, or both.
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Particle Tree. The asterisk indicates a required field.
Protocol Name
Enter the protocol from which the data is derived.
Protocol Version
Enter identifying version information for the protocol version from which the data is derived.
Description
Enter information not covered by other fields on the form.
Instrument Information
Instrument Type
Identify the kind of instrument used to obtain data.
Other Instrument Type
Enter Instrument Type that is not listed in the drop-down list.
Instrument Manufacturer
Select the manufacturer of the instrument used to obtain the data.
Other Instrument Manufacturer
Enter Other Instrument Manufacturer that is not listed in the drop-down list.
Description
Enter information about instrumentation not covered by other fields.
Purity Image
 
Number of Images
Enter the number of Purity Images.
Update Images
Note: If you enter a number greater than “0”, complete all of the additional information for the particle, then click the Update Images button at the bottom of the page. This extends the page with the corresponding number of additional fields so you can enter information for each unit.
Extended Page: Graph Data
 
Graph Type
Select from the drop-down list the appropriate graph type:
•
Image
•
Graph
Characterization File Name
This field remains empty until you select the file to be uploaded. To do so, click the Load File button, opening the Load Characterization File page. For more information about completing the form, see Loading Characterization Files.
3.
Once you have entered the appropriate information on the form, click Submit.
OR
4.
Click Reset to clear the fields on the form.
