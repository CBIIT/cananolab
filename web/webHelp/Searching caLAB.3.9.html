Searching Nanoparticles
Note:
In the conventional sense, the term “sample” can refer to either of two entities in caLAB: 1) A “Sample” is identified by a vendor, a sample accession date, etc; this is not the sample (identified when creating an aliquot) whose search this topic describes; and 2) a “Nanoparticle” sample is annotated with research and other characterization data.
In caLAB and in this online help, the first sample is identified as Sample and the second sample is identified as Nanoparticle. This topic describes performing a search for a “Nanoparticle”.
To launch a search for a specific nanoparticle, follow these steps:
1.
Select Search > Search Nanoparticles. This opens the Search Nanoparticle page.
2.
 
3.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, in searching for sample NCL-8-234, you could enter NCL-8*.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search. If you click the Search button having defined no parameters, all of the caLAB nanoparticles will be listed in the Search Results.
Nanoparticle Search Options

Particle Source
Select from the drop-down list the source of the nanoparticle.
Particle Type
Select from the drop-down list the name of the Nanoparticle Type being searched. Example: Carbon Nanotube.

Select from the drop-down list the category of function to which the nanoparticle contributes.

Characterization Type
Click from the displayed list the characterization type hypertext link that applies to the nanoparticle to be searched.
 
When you click a hypertext link, Characterization sub-categories display in the right-hand pane of the row. Select one or more of the items in the list. (CTRL + shift to select more than one.)
Keywords
Enter keywords that can be used for the nanoparticle search. These would correspond to keywords entered on the General Information form when annotating a nanoparticle.
4.
Click Search to execute the search. To clear all fields, click the Reset button.
For information about search results, see Nanoparticle Search Results.
