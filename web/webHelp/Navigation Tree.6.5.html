Particle Diagnostic Reporting Function
The Navigation Tree Function heading enables you to add new Diagnostic Reporting function details as corresponding annotation data or edit an existing function.
When you are adding a new function, all subcategories open a form with the fields described in The Particle Function Table.
Once the new function is added, you can edit the function by clicking the hyperlinked name under Diagnostic Reporting in the Navigation Tree. Use The Particle Function Table to edit the existing entries of the function.
For more information about nanoparticle functions, see Function Category.
