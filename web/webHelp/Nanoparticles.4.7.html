Nanoparticle Search Results
If you click the Search button having defined no parameters, all of the caNanoLab Nanoparticles will display in the search results.
Nanoparticle search looks for a nanoparticle based on annotations that have been added or characterization or data files that have been uploaded. Nanoparticle Search Results display in the browser window in table format. Column headings are appropriate for the nanoparticle for which you searched. They are described in the following table. Columns are sortable by clicking the column headings.
Nanoparticle Search Results Columns
Description
Particle Sample Name
The fully qualified name of the nanoparticle hit
Particle Source
The name of the source of the original nanoparticle. For instructions for each type, see NCL.
Particle Composition
The term for the nanoparticle composition. Examples: antibody; dendrimer.
Particle Functions
A list of one or more terms displaying functions that have been entered for the corresponding Particle ID. Example: targeting
Particle Characterizations
A list displaying characterization annotations that have been entered for the corresponding Particle ID. Example: oxidative stress
Particle Location
The location of the nanoparticle data service
To open detailed information about any search hit, click on its hypertext link in the Particle ID column (first). caNanoLab opens to the Updating Nanoparticle Sample page. Details for only one nanoparticle display at a time.
For information about configuring a nanoparticle search, see Searching Existing Nanoparticles.
