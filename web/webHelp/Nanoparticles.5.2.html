Nanoparticles Overview
When you click the Nanoparticles option on the caNanoLab menu bar, the Manage Nanoparticle Home page appears. The Nanoparticle Links panel on the Manage Nanoparticle Home page provides the following options:
•
Annotate a New Nanoparticle: This takes you to a form for annotating a nanoparticle in the caNanoLab database to which no annotations have yet been added. For more information, see Submitting a Nanoparticle - General Information.
•
Search Existing Nanoparticles: This takes you to a form for launching a search for an existing nanoparticle to which annotations have already been added. For more information, see Searching Existing Nanoparticles.
For more information about nanoparticles, see Description of a Nanoparticle.
