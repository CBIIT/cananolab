Deleting a Characterization
To delete a characterization for a selected particle, follow these steps:
This only deletes the characterization(s) for the selected particle. If it (they) has (have) been copied for other particles, it will not be deleted.
1.
From the Navigation Tree, click a Delete button next to the subcategory of a characterization you want to delete.
The Delete Characterizations page opens listing all the current characterizations for the category.
2.
To select a characterization, check the box.
To cancel the deletion, click the Reset button.
3.
To delete the selected characterization, click the Delete button.
Deleted characterizations are placed in the particle archive for history purposes.
