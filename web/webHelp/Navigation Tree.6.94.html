Displaying a Summary View of In Vitro Cytotoxicity Characterizations
If there are multiple In Vitro Characterizations added under a subcategory in the Navigation Tree, you can display a summary view of all the characterizations in one table. With additional icons, you can print the table or save the table to a spreadsheet file. You can also display and print the individual detail views of the categoryâ€™s characterizations in one table. Each In Vitro Cytotoxicity Characterization is defined briefly:
Caspase 3 Activity
Cell Viability
To display all the characterizations in one table, follow these steps:
1.
Under Cytotoxicity, click a subcategory such as Cell Viability. The appropriate In Vitro Characterization page opens listing all the characterizations in one table, one characterization per row.
