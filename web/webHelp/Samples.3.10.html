Aliquot Naming Conventions
In caNanoLab, aliquots are named automatically when they are created, using the following convention:
prefix (NCL)-sequence # -lot ID. Example: NCL-8-4-3.
•
The first portion of the name indicates the sample from which the aliquot is created (NCL-8).
•
The second portion of the name indicates the aliquot number from that sample (NCL-8-4 [#4 aliquot from sample NCL-8]).
•
If you create an aliquot from an existing aliquot, the third portion of the name indicates the aliquot number from that aliquot (NCL-8-4-3, [aliquot #3 created from aliquot #4 created from sample NCL-8]).
•
If you create more than one aliquot from the same sample, they are numbered consecutively as in NCL-8-1, NCL-8-2, NCL-8-3, etc.
