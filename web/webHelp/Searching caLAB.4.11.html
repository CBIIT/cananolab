Searching Nanoparticles
In the conventional sense, the term “sample” can refer to either of two entities in caNanoLab. This topic describes performing a search for a “Nanoparticle” sample type, that is, a nanoparticle sample to which annotations have been added. For more information, see caNanoLab Sample Overview.
This search feature is accessible even to users for whom other search features are restricted.
To launch a search for a specific nanoparticle, follow these steps:
1.
Select Search > Nanoparticles. This opens the Search Nanoparticle page.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab nanoparticles will be listed in the Search Results.

Characterization Type
Click from the displayed list on the left, the characterization type hypertext link that applies to the nanoparticle to be searched.
When you click a hypertext link, Characterization sub-categories display in the right-hand pane of the row. Select one or more of the items in the list. (CTRL + shift to select more than one.)
Summary/Description
Enter one word per line that summarizes or describes the characterization or uploaded file. Select the appropriate button for the search target.
Keywords
Enter keywords, one word per line, that can be used for the nanoparticle or characterization search. These would correspond to keywords entered on the Nanoparticle General Information form when annotating a nanoparticle. Select the appropriate category for the search, nanoparticle or characterization..
3.
Click Search to execute the search. To clear all fields, click the Reset button.
For information about search results, see Nanoparticle Search Results.
