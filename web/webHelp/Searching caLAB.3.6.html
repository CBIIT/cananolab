Sample Search Results
Sample Search Results display in the browser window in table format. Column headings are appropriate for the sample type for which you searched.
 
Sample Search Results Columns
Description
Select
Select one or more of the search hits for which you want to view details. Click the View Details button for detailed information about the samples. Click Show Aliquots to view the associated aliquots.
Sample ID
The fully qualified name of the nanoparticle to be searched. Add an asterisk for a wild-card search.
Container Name
The name of the container storing the sample.
Sample Accession Date
The date the sample was accessioned into the laboratory.
Sample Type
The name of the sample type being searched. Example: Carbon Nanotube.
Sample Location
The location of the Sample.
Sample Submitter
The name of the person who submitted the sample.
For any hit for which you want more information, select the corresponding radio button. Click the View Details button to open a sample details page. Click Show Aliquots to open details for aliquots created from the selected sample.
For more information about configuring a sample search, see Searching Samples..
