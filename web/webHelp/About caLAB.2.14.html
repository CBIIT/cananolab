Manage Aliquots
Two options display when you select the Inventory > Manage Aliquots menu option in caNanoLab.
Creating a New Aliquot
This takes you to a form for entering information for a new sample you want to submit. For more information, see Creating A New Aliquot.
Select an Existing Aliquot--
This takes you to a form for launching a search for an existing aliquot. For more information, see Searching Aliquots.
