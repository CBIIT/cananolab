Searching Nanoparticle Reports
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle report results from the Login page.
The Search Report feature allows all caNanoLab users to find and view nanoparticle reports. This feature is accessible even to users for whom access to nanoparticles in caNanoLab is restricted.
To launch a search for a specific nanoparticle, follow these steps:
1.
Click the Reports caNanoLab menu option. On the Manage Report Home page, click the Search Existing Reports link. This opens the Search Report form.
At the top of the page is the Grid Search link that enables you to search for nanoparticles on remote hosts. For more information, see Performing a Remote Nanoparticle Search.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab reports will be listed in the Search Results.

Enter the name of the report.

The clear the forms and start again, click Reset.
3.
To launch the search, click Search.
For information about search results, see Nanoparticle Reports Search Results.
