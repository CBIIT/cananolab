The Navigation Tree > Sample Composition
Sample Composition
When you click the Navigation Tree Sample Composition link in the left sidebar, the Sample Composition Properties subcategories display:
Nanoparticle Entity
Functionalizing Entity
Chemical Association
Composition File
You can click the link again to collapse the subcategories. If annotation information is filled out for the nanoparticle, the annotations are listed under the Composition Categories.
You may have the option to add or delete function details with the Add and Delete buttons. If you do not have certain security privileges, the Add and Delete buttons are hidden, and you have read-only access to the existing items.
See also Adding Sample Composition Annotations.
