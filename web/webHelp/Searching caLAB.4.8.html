Searching Aliquots
To launch a search for a specific aliquot, follow these steps:
1.
Select Search > Search Aliquots. This opens the Search Aliquot page.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, in searching for aliquot NCL-21-1-4, you could enter NCL-21*.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab aliquots will be listed in the Search Results.
Aliquot Search Options
Description
Aliquot ID
Enter the fully qualified name of the Aliquot to be searched. Add an asterisk for a wild-card search.
Sample Type
Select from the drop-down list the name of the Sample Type being searched. Example: Emulsion, Liposome.
Source
Select from the drop-down list the name of the Source from which the aliquot originated. Example: XYZ institute.
Source ID
Select from the drop-down list the unique identifier for each aliquot source.
Date Accessioned
Click the two calendars to define the range of dates during which the all aliquots were accessioned in the lab.
Creator
Select from the drop dlown list the name of the person who created the aliquot.
Storage Location
Enter or select the location in which this aliquot is stored. Example: Freezer1
3.
Click Search to execute the search. To clear all fields, click the Reset button.
