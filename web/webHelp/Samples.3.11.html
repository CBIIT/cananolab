Creating A New Aliquot
To create aliquots, you must have researcher privileges.
You must create one or more samples before you create aliquots of those samples. This follows the same workflow tasks as would be performed in a laboratory. (Aliquots are normally created in batches, not singley.) For more information see Creating a New Sample.
 
To create an aliquot, follow these steps:
1.
Click the Samples caNanoLab menu option. On the Manage Sample/Aliquot Home page, click the Create a New Aliquot link. This opens the Create Aliquot form.
2.
Enter or select the Description information described in the following table. Lists associated with drop-down menus are predefined in caNanoLab. Items that have an asterisk are required fields.
The characters [-?-] indicate you can enter text, or you can select from the drop-down list.
Name
Description
Sample ID* or Aliquot ID*
Identify the aliquot source by selecting the appropriate radio button:
•
Sample Container
•
Aliquot Container
Select from the corresponding drop-down list the name of the sample or aliquot from which the new aliquot is being created. This is a sample or aliquot that must have previously been created in caNanoLab. (The IDs are automatically generated in caNanoLab. For more information see Aliquot Naming Conventions.)
Note: If both Sample ID and Aliquot ID are selected from the drop down menu, the system will use Aliquot ID and ignore Sample ID.
Number of Aliquots (Update Aliquots button)
Enter the number of aliquots to be created in this batch of aliquot creation.
Note: If you enter a number greater than “1”, complete all of the additional information for the aliquot, then click the Update Aliquots button at the bottom of the page. That way, the information you enter becomes part of your container template and you don’t have to re-enter it again. This is particularly helpful when you are inserting a large number of aliquot containers. See Step #3.
3.
To create an aliquot template for the sample, enter or select the Template Aliquot information in the following table. The information you specify will be the default for all aliquot added to this sample.
Name
Description
Container Type*
Select or enter the Container Type in which the aliquot is stored from the pre-populated drop-down list. Options: Tube, Vial or Other.
Quantity
In the first text box, enter the weight of the aliquot that is stored in this container. In the drop down list, select the unit used.
Concentration
In the first text box, enter the value of the Concentration of the aliquot in the current container. In the drop down list, select its associated measurement unit.
Note: If Concentration is entered, Diluents/Solvents must also be entered and vice versa.
Volume
In the first text box, enter the value of the Volume in each container in the first text box. In the drop down list, select its associated measurement unit.
Diluents/Solvent
Diluents: Name the liquid used to dilute the liquid sample.
Solvent: Name the solvent used dissolve the solid sample.
Note: If Diluents/Solvents is entered, Concentrationmust also be entered and vice versa.
How Created
Select one of the relevant options for the Aliquot you are creating. In caNanoLab 1.3, the options for creation method are:
•
Solubilized
•
Lyophilized
View Protocol
Click the button to open protocol information describing the laboratory procedure for creating the aliquot. View Protocol opens the document based on the selection in the How Created drop down menu.
Note: An available protocol must be submitted to caNanoLab for it to be incorporated into and available in caNanoLab. For more information, see Submitting a Protocol.
Storage Conditions
Enter the temperature and other variables required to adequately store the aliquot in the container.
Storage Location
Enter or select the location (Room, Freezer, Shelf, Box) in which this container is stored. Example: Freezer1, Box 12.
General Comments
Enter additional comments that are associated with this particular aliquot container.
4.
After completing the Aliquot template, you can add Aliquot forms to the sample.
1.
To add an Alquot form, click the red Update Aliquots button.
The number of forms added is based on the Quantity entered in the Description table.
2.
Modify each form appropriately to customize the description for each Aliquot. You can edit the aliquots you have created. See Editing an Aliquot for more information, however, once entered into the database, an aliquot can no longer be edited.
To clear the entries, click the Reset button.
3.
Click the Submit button after defining the sample(s) appropriately.
After you click the Submit button, if you have failed to enter appropriate information for particular fields, such as the solvent when concentration is entered or the container volume unit, you will receive an error message. You must correct the error before your submission is accepted.
The detailed information you entered displays in table format. A separate table displays for each container you described in your entry.
