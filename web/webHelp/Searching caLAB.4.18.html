Protocol Search Results
Note: If you click the Search button having defined no parameters, all of the protocols located in the search display in the search results.
Once you have submitted a search for a protocol (Searching Protocols), search results display in the caNanoLab screen. Results show pertinent protocol information and a URI to each report. Click the Protocol Name URI to locate and open a protocol file.
The table columns are sortable; click a column title to sort it.
