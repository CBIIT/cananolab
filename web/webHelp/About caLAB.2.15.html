Creating A New Aliquot
Before an Assay Run can be performed in caNanoLab, it is necessary to create one or more samples, then create aliquots of those samples, following the same workflow tasks as would be performed in a laboratory. (Aliquots are normally created in batches, not singly.) For more information see Creating a New Sample.
 
Note: To create aliquots, you must have NCL researcher privileges.
To create an Aliquot, follow these steps:
1.
To create an aliquot, click Inventory > Manage Aliquots. This displays two links; click Create a New Aliquot. This opens the Create Aliquot form.
2.
Enter or select the appropriate information about the aliquot you are creating in the text entry fields and drop-down menus (described in the following table). Lists associated with drop-down menus are predefined in caNanoLab. Items that have an asterisk are required fields.:
Create Aliquot Text Fields
Description
Sample ID* or Aliquot ID*
Identify the aliquot source by selecting the appropriate radio button: Sample Container or Aliquot.
Select from the corresponding drop-down list the name of the sample or aliquot from which the new Aliquot is being created. This is a sample or aliquot that must have previously been created in caNanoLab. (The IDs are automatically generated in caNanoLab. For more information see Aliquot Naming Conventions.)
Note: If both Sample ID and Aliquot ID are selected from the drop down menu, the system will use Aliquot ID and ignore Sample ID.
Number of Aliquots (Update Aliquots button)
Enter the number of Aliquots to be created in this batch of Aliquot creation.
Note: If you enter a number greater than “1”, complete all of the additional information for the Aliquot, then click the Update Aliquots button at the bottom of the page. That way, the information you enter becomes part of your container template and you don’t have to re-enter it again. This is particularly helpful when you are inserting a large number of aliquot containers. See Step #3.
Template Aliquot
Container Type*
Select the Container Type in which the aliquot is stored from the pre-populated drop-down list. Options: Tube, Vial or Other.
Other
If Other [Container Type] is selected, enter the name of the Container Type that is used to store the Aliquot.
Quantity
In the first text box, enter the weight of the aliquot that is stored in this container. In the drop down list, select the unit used.
Concentration
In the first text box, enter the value of the Concentration of the aliquot in the current container. In the drop down list, select its associated measurement unit.
Note: If Concentration is entered, Diluents/Solvents must also be entered and vice versa.
Volume
In the first text box, enter the value of the Volume in each container in the first text box. In the drop down list, select its associated measurement unit.
Diluents/Solvent
Diluents: Name the liquid used to dilute the liquid sample.
Solvent: Name the solvent used dissolve the solid sample.
Note: If Diluents/Solvents is entered, Concentrationmust also be entered and vice versa.
How Created
Select one of the relevant options for the Aliquot you are creating. In caNanoLab 1.1, the options for creation method are:
•
Solubilized
•
Lyophilized
View Protocol
Click the button to open protocol information describing the laboratory procedure for creating the aliquot. View Protocol opens the document based on the selection in the How Created drop down menu.
Note: An available protocol must be submitted externally to the caNanoLab team for it to be incorporated into and available in caNanoLab. In caNanoLab 1.1, available protocols have been provided by the NCL.
Storage Conditions
Enter the temperature and other variables required to adequately store the aliquot in the container.
Storage Location
Enter or select the location in which this container is stored. Example: Freezer1
General Comments
Enter additional comments that are associated with this particular aliquot container.
3.
After clicking the Update Aliquots button, you can edit the aliquots you have created. See Editing an Aliquot for more information. Once entered into the database, an aliquot can no longer be edited.
4.
Click the Submit button after defining the sample(s) appropriately.
After you have clicked the Submit button, if you have failed to enter appropriate information for particular fields, such as the solvent when concentration is entered or the container volume unit, you will receive an error message.You must correct the error before your submission is accepted.
