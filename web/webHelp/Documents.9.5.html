Searching Nanoparticle Documents
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle documents from the Login page.
The Search Documents feature looks for submitted publications or reports associated with one or more nanoparticles. This feature allows all caNanoLab users to find and view nanoparticle publications or reports, even users for whom access to nanoparticles in caNanoLab is restricted.
To launch a search for a specific document, follow these steps:
1.
Click the Documents caNanoLab menu option. On the Manage Documents home page, click the Search Existing Documents link. This opens the Search Documents form..
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Asterisks indicate required fields.
When defining search criteria, there are no “required” fields. You can add only those parameters that you consider essential to the search. Add an asterisk for searching wildcards.
If you click the Search button having defined no parameters, all of the caNanoLab documents will be listed in the Search Results.
