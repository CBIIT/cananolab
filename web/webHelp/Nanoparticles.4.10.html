Adding Documents to a Nanoparticle
Publications and reports can be associated with nanoparticles using two caNanoLab features.
1.
If you work from the Documents menu option, you can associate multiple nanoparticles with a publication or report. See Documents Overview for more information
2.
If you add documents to an open nanoparticle, you can use the copy feature to associate the document with other nanoparticles. See Submitting a Publication for more information.
