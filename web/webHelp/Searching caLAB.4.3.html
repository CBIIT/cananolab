Searching a Workflow
The Search Workflow feature in caNanoLab searches a workflow for assay run and associated aliquot information based on query input.
To launch a search for one or more specific components in an assay, follow these steps:
1.
Select Workflow > Search Workflow. This opens the Search Workflow page.
2.
Enter text or select options as appropriate (described in the following table). In any text field, you can add an asterisk for a wild-card search. For example, if the aliquot specified in the search is NCL-23-4-2, you could enter NCL-23*.
When defining search criteria, there are no "required" fields. You can specify only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, components of all of the caNanoLab runs will display in the Search Results.
Workflow Search Text Fields
Description
Assay Name
Enter the name of the assay that is used in the workflow being searched. Add an asterisk for a wild-card search.
Assay Type
Select from the drop-down list the type of experiment used. Options are preconfigured for the workflow being searched.
Assay Run Date
Click the two calendars to define the range of dates for the search target.
Aliquot ID
Enter the unique ID of the Aliquot that is associated with the Workflow to be searched.
File Name
Enter the name of the file(s) that are used as input or result (output).
File Submission Date
Click the two calendars to define the range of dates during which files were uploaded.
File Submitter
Select in the drop-down menu the name of the person who uploaded the file(s).
Join Search Criteria By
Select AND or OR logic before performing the Workflow Search.
 
Note:
With the AND criterion selected, a search that satisfies all the entered and checked criteria is performed. With OR selected, a search that satisfies any one of the criteria is performed.
Additional Filtering of Target Data
Input Files?
Check the box if you want only input files.
Output Files?
Check the box if you want only output files.
Exclude Masked Aliquots?
Check the box to exclude aliquots currently masked in the system when searching the workflow. Leave the box clear to include all aliquots.
Exclude Masked Files?
Check the box to exclude files currently masked in the system in the search. Leave the box clear to locate all files.
3.
Click Search to execute the search. To clear all fields, click the Reset button.
For information about search results, see Workflow Search Results.
