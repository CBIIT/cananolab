Searching Existing Samples
This function is limited to users belonging to a specific research group. For more information, see Permissions and Visibility.
In the conventional sense, the term “sample” can refer to either of two entities in caNanoLab. For more information, see Description of a Sample. This topic describes performing a search for a “Sample”. For information about searching for “Nanoparticles”, see Searching Existing Nanoparticles.
To search for an existing sample, follow these steps:
1.
Click the Samples caNanoLab menu option. On the Manage Sample/Aliquot Home page, click the Search Existing Samples link. This opens the Search Samples form.
2.
Enter or select the Search information described in the following table. In any text field, you can add an asterisk for a wild-card search. For example, in searching for sample NCL-8-234, you could enter NCL-8*.
There are no required fields. You can add only those parameters that you consider essential to the search.
If you click the Search button having defined no parameters, all of the caNanoLab samples will be listed in the Search Results.
Name
Description
Sample ID
Enter the fully qualified name of the Sample to be searched. Add an asterisk for a wild-card search.
Sample Type
Select from the drop-down list the name of the Sample Type being searched. Example: Dendrimer, Liposome.
Source
Select from the drop-down list the name of the Source from which the sample originated. Example: XYZ institute.
Source ID
Select from the drop-down list the unique identifier for each source.
Date Accessioned
Click the two calendars to define the range of dates during which the samples were accessioned in the lab.
Creator
Select from the drop-down list the name of the person who submitted the Sample.
Storage Location
Enter or select the location (Room, Freezer, Shelf, Box) in which this container is stored. Example: Freezer1, Box 12.
To clear all fields and start again, click the Reset button.
3.
Click Search to execute the search.
