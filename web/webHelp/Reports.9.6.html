Report Update Page
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle report results from the Login page.
The Report Update page opens when you click any of the hypertext file name links in the Nanoparticle Reports Search Results page.
The page that displays is the same Report File page on which the report was submitted into caNanoLab. Details are described in the following table. Asterisks indicate required fields.
Nanoparticle Report Criteria
Description
Upload Report File
or
Enter Report URL
Click the Browse button to navigate to the file to be associated with the report, or enter the URL where the report is located.
Submitted Report
This category displays only if you are viewing/editing an existing report. Click the hypertext report name to open the report.
Report Category
Select the report category from the drop-down menu.
Report File Title
Enter the report title.
Nanoparticle Sample Name
Select one or more nanoparticle sample names, to be the subject of the report, from the list.
Report File Description
Enter information not covered by other fields on the form.
Comments
Enter additional appropriate comments about the report.
Visibility
The term, Visibility, is used to describe the access to objects granted to logged in users. Select the group to which this report is to be visible.
