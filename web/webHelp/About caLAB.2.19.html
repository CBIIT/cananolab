Using an Aliquot
When you are creating an Assay Run, and click the In folder in the Run Tree, a Use Aliquots menu option displays on the Run Details - Workflow General Information page. Click Use Aliquots to open the Use Aliquot dialog box where you can specify the aliquot(s) for this assay run. To use an aliquot, you must include it as part of an Assay Run. For more information, see Creating an Assay Run.
