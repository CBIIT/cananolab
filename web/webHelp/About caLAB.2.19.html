Editing an Aliquot
Editing an aliquot is part of the process of creating an aliquot.
To edit an aliquot, follow these steps:
1.
Complete the necessary steps for creating an aliquot, through step 3 in Creating A New Aliquot.
2.
Click the Update Aliquots button before submitting the aliquot(s) to the caLAB database.
At this point, for each container you described in your entry, a box displays naming the aliquot, (which has not yet been entered into the caLAB database). Aliquots are named automatically by caLAB. For more information, see Aliquot Naming Conventions.
3.
Click on the hypertext link in the box to open an Edit Aliquot page displaying the corresponding details for the Aliquot.
You can edit any aliquot information here.
4.
Click Submit on this page, returning you to the Create Aliquot page where the edited information now displays in the individual aliquot boxes.
5.
Click Submit on this page to enter these aliquots, based on a particular Sample ID, into the caLAB database. A message verifies when the aliquots are sucÂ­cessfully submitted into the database.
Once entered into the database, the aliquot can no longer be edited. An aliquot can be masked, however. For more information, see Masking an Aliquot.
