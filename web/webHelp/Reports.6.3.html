Submitting a Nanoparticle Report
This feature allows you to upload a final nanoparticle report for a vendor or researcher. The report can contain information about multiple particles.
To upload a nanoparticle report, follow these steps:
1.
Click the Reports caNanoLab menu option. On the Manage Report Home page, click the Submit a New Report link. This opens the Submit Nanoparticle Report form.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table.
3.
On the Submit Nanoparticle Report form that opens, select or complete the following fields.
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point,your entry is sandwiched by [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear
Nanoparticle Report Criteria
Description
Upload Report File
or
Enter Report URL
Click the Browse button to navigate to the file to be associated with the report, or enter the URL where the report is located.
Report Category
Select the report category from the drop-down menu.
Report File Title
Enter the report title.
Nanoparticle Sample Name
Select one or more nanoparticle sample names, to be the subject of the report, from the list.
Report File Description
Enter information not covered by other fields on the form.
Comments
 
Visibility
The term, Visibility, is used to describe the access to objects granted to logged in users. Select the group to which this report is to be visible.
To clear the fields and start again, click Reset.
4.
Once you have entered the appropriate information on the form, click Submit.
