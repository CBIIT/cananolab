Using caNanoLab Online Help
The online help explains how to use all of the features of the caNanoLab portal. To access online help, use any of the following methods.
You can access online help without being logged into caNanoLab.
•
To open the complete online help project, click the Help menu option.
•
Click the Help link at the top of each page to open a context-sensitive topic. Context-sensitive help displays information that corresponds to the page from which help was opened.
See the following tips for locating topics of interest, navigating the help, and printing a topic.
