Sample Search Results
Note:
If you click the Search button having defined no parameters, components of all of the caNanoLab samples will display in the search results.
Sample Search Results display in the browser window in table format. Column headings are appropriate for the sample type for which you searched; they are described in the following table. Columns are sortable.
Sample Search Results Columns
Description
Select
Select one or more of the search hits for which you want to view details. Click the View Details button. Click Show Aliquots to view the associated aliquots.
Sample ID
The fully qualified name of the sample hit.
Container Name
The name of the container storing the sample.
Sample Accession Date
The date the sample was accessioned into the laboratory.
Sample Type
The name of the nanoparticle sample type. Example: Carbon Nanotube.
Sample Location
The storage location of the sample.
Sample Submitter
The name of the person who submitted the sample.
For any hit for which you want more information, select the corresponding radio button.
