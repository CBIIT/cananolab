Nanoparticle Reports Search Results
If you click the Search button having defined no parameters, all of the caNanoLab Nanoparticle Reports will display in the search results.
Nanoparticle Report Search Results display in the browser window in table format. Column headings are appropriate for the report(s) for which you searched; cell entries were defined when the report was submitted. Columns are sortable. Column headings are described in the following table.
Report Search Results Columns
Description
Report Title
The name given to the report
Report Type
The report type you selected when you created the report.
Report Description
The description of the report
For any hit, click on the hypertext link in the Report Title column (first) to open the associated report or file.
