Welcome to caNanoLab Home Page
Once you log into caNanoLab, the Welcome to caNanoLab Home page opens. The flowchart on the Home page illustrates a general workflow to use in caNanoLab. For more information about this workflow diagram, see caNanoLab Functionality Overview.
The caNanoLab workspace comprises the menu bar and the blue panel on the left side of the browser.
•
All users can access caNanoLab Quick Links (hypertext links in the blue panel on the left sidebar of the browser). These link to the National Cancer Institute (NCI) website, the NCI Center for Bioinformatics website, and the Nanotechnology Characterization Laboratory (NCL) website.
•
The caNanoLab menu options are displayed according to assigned permissions. See Permissions and Visibility. Options which may be available are described as follows:
When you display a caNanoLab page some fields may contain a question mark like [-?-]. This indicates that you can either make a selection from the drop-down menu or type an entry when [-?-] displays.
 
Menu Options
Description
Samples
Add new and search for existing Samples and add new and edit and search for existing Aliquots. For more information, see Managing Samples and Aliquots.
Protocols
Add new and search for existing protocols. For more information, see Managing Protocols.
Nanoparticles
Annotate new nanoparticles. Search for existing nanoparticles or perform a search of nanoparticles in remote hosts. For more information, see Managing Nanoparticles.
When you add a nanoparticle to caNanoLab with the Nanoparticles option, the Navigation Tree fills in the remainder of the blue panel. For more information about the Navigation Tree, see The Navigation Tree.
Reports
Submit new and search for existing nanoparticle reports. For more information, see Submitting Reports.
Administration
This feature is only available to users with assigned administrative permission.
Access the Common Security Module (CSM). caNanoLab allows authorized administrators to manage user accounts, groups, roles, and protection elements.
Home
Return to the Home page and Functional Overview. For more information, see caNanoLab Functionality Overview.
Help
Open full help features. For more information see Using caNanoLab Online Help.
Logout
Logout of caNanoLab and return to the caNanoLab login page.
