Creating a New Sample
If you do not have certain security privileges, the Creating a New Sample link does not appear on the Manage Sample/Aliquote Home page.
In a laboratory setting, when a new sample comes in from an outside source, you would log it in and store it in a repository. For using it, you would dissolve or dilute the sample, then use it to create aliquots for use at the bench following customized protocols. For more information, see Description of a Sample.
In caNanoLab, if you have researcher privileges, you can “create” a sample, by recording a physical description of the source sample and its origin. Then you “create” associated aliquots by describing the process by which the source sample is dissolved and/or diluted and otherwise prepared for bench use. You can also view associated protocols that have been previously implemented into caNanoLab.
To create a Sample, follow these steps:
1.
Click the Samples caNanoLab menu option. On the Manage Sample/Aliquot Home page, click the Create a New Sample link. This opens the Create Sample form.
2.
Enter or select the Sample Information described in the following table. Lists associated with drop-down menus are predefined in caNanoLab. Items that have an asterisk are required fields.
The characters [-?-] indicate you can enter text, or you can select from the drop-down list.
 
Name
Description
Sample ID Prefix*
Represents the fully qualified name of the Sample. This field is pre-determined by the system as the sample is created, however, the field can also be edited. Example: NCL-8. (NCL is the prefix for the NCL lab.) For more information, see Aliquot Naming Conventions.
Sample Type
Select from the drop-down menu the name of the Sample Type to which the current sample belongs. Example: Dendrimer, Liposome.
Description
Enter additional comments relevant to the sample creation process.
Source
Enter or select the name of the originating source for this sample.
Source ID
Enter the unique ID that was assigned by the organization where the sample originated.
Date Received
Click the calendar to locate the date on which the sample was received by the lab from the source.
SOP
Select the Standard Operating Procedure (SOP) type that was used by the lab to generate the sample. You can leave the text field blank, or select between Original or Testing.
Note: SOPs can be submitted to caNanoLab for inclusion in the software as Protocols.
Lot ID
Enter the identifier of the lot such as the lot number for this Sample.
If you enter a Lot ID, the ID is appended to the Sample ID.
Lot Description
Enter a description about the lot to which this Sample belongs.
Solubility
Enter the value representing how much of a given substance dissolves in a particular solvent.
General Comments
Enter additional information about the sample creation not covered by other fields in the form.
3.
To create a container template for the sample, enter or select the Container Information described in the following table. The information you specify will be the default for all containers added to this sample.
 
Name
Description
Container Type*
Select or enter the Container Type in which the sample is stored in the pre-populated drop-down list. Options: Tube, Vial or Other.
Quantity
In the first text box, enter the weight of the sample that is stored in this container. In the drop down list, select the unit used.
Concentration
In the first text box, enter the value of the Concentration of the sample in the current container. In the drop down list, select its associated measurement unit.
Note: If Concentration is entered, Diluents/Solvents must also be entered and vice versa.
Volume
In the first text box, enter the value of the Volume in each container in the first text box. In the drop down list, select its associated measurement unit.
Diluents/Solvent
Diluents: Name the liquid used to dilute the liquid sample.
Solvent: Name the solvent used to dissolve the solid sample.
Note: If Diluents/Solvents are entered, Concentration must also be entered and vice versa.
Safety Precautions
Enter safety precautions that are worth noting.
Example: The sample is Toxic.
Storage Conditions
Enter the temperature and other variables required to adequately store the sample in the container.
Storage Location
Enter or select the location (Room, Freezer, Shelf, Box) in which this container is stored. Example: Freezer1, Box 12.
Comments
Enter additional comments that are associated with this particular sample container.
4.
Once you complete the container template, you can add container forms to the sample.
1.
To add a container form, click the red Add Container link.
A new container form is added.
2.
Modify each form appropriately to customize the description for each container.
To clear the entries, click the Reset button.
3.
Click the Submit button after defining the sample(s) appropriately.
After you click the Submit button, if you have failed to enter appropriate information for particular fields, such as the solvent when concentration is entered or the container volume unit, you will receive an error message. You must correct the error before your submission is accepted.
The detailed information you entered displays in table format. A separate table displays for each container you described in your entry.
Samples are used to create aliquots. For more information, see Creating A New Aliquot.
