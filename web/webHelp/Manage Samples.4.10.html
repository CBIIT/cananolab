Performing a Remote Sample Search
When you launch a sample search, you can specify at the top of the search form the caNanoLab data service sites you want to search against. You can search for publicly available samples from all active caNanoLab data services connected to the caBIG® grid (caGrid™). For more information about caGrid, see https://cabig.nci.nih.gov/workspaces/Architecture/caGrid.
The remote search result pages are identical to the public search result pages. See Searching Existing Samples for more information.
