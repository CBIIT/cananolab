Launching a caLAB Search
Access to features and objects in caLAB is dependent upon the permissions assigned registered users. The caLAB 1.0 allows you, with exceptions based on your privileges, to perform the following searches:
Run Search
A Run Search, to identify Runs that can be included in a Workflow, is part of executing a Workflow. See Searching Runs.
Workflow Search
A Workflow Search looks for Run information such as a file, an aliquot, etc. See Searching a Workflow
Sample Search
A Sample Search looks for samples that may be recorded in caLAB. Samples are identified by source vendors, date of receipt, storage location, etc. See Searching Samples.
Aliquot Search
An Aliquot Search looks for aliquots, identified by the source from which the aliquot is derived, the name of the person who submitted the aliquot, storage location, etc. See Searching Aliquots.
Nanoparticle Search
A Nanoparticle search looks for a nanoparticle based on annotations that have been added such as characterization information. See Searching Nanoparticles
Reports Search
A Reports search looks for submitted reports associated with multiple nanoparticles. This feature is accessible even to users for whom access to nanoparticles in caLAB is restricted. See Searching Reports.
