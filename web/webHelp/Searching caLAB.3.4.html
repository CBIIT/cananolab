Workflow Search Results
Note:
If you click the Search button having defined no parameters, all of the caLAB workflow information will display in the search results.
Workflow search results display in the browser window, summarizing the Assay Run information in table format. Column headings are similar to the text fields you defined in the search criteria.
To view the details of a search hit or to download the associated file, click one of the displayed file names. Each is a hypertext link.
Workflow Search Result Fields
Description
File Name
The name of the file associated with the Assay Run results
Assay Type
The type of experiment used.
Assay Name
The name of the assay used in the workflow being searched.
Assay Run Name
The name assigned to the Assay Run corresponding to the workflow
Assay Run Date
The date for the Assay Run corresponding to the workflow
In/Out File
The type (input, ouput) of the submitted file
File Submission Date
The date that the corresponding search hit was submitted.
File Submitter
The name of the person who uploaded the file(s).
File Status
Status of the named file.
Aliquot ID
IDs of the Aliquots associated with the Workflow search hit.
Aliquot Status
Status of the associated Aliquot
For more information about configuring a workflow search, see Searching a Workflow.
