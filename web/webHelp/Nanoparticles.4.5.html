Submitting/Updating a Nanoparticle Sample
Update Nanoparticle Sample: If you perform a nanoparticle search, on the Nanoparticle Search Results page, and click the hypertext Particle Sample Name in the first column, this Update Nanoparticle page opens. On this page, you can review and/or edit the information about the particle that was defined when it was submitted. See Update Nanoparticle Sample for more information.
Submit Nanoparticle Sample: To enter general information about the nanoparticle sample, follow these steps:
1.
Click the Nanoparticles caNanoLab menu option. On the Manage Nanoparticle Home page, click the Submit a New Nanoparticle link. This opens the Submit Nanoparticle-General Information form.
2.
Select or enter the appropriate information in the text fields and drop-down list. Fields are described in the following table. Asterisks indicate required fields.
Some fields may contain the Other option. Click Other to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear.
Name
Definition
Nanoparticle Sample Name*
Enter a sample name of your choice.
 
Nanoparticle Sample Source*
Identify in the drop-down list the site that is the source of the sample.
Keywords
Add any keyword(s) you want associated with the selected nanoparticle, one keyword per line. You can enter something not necessarily captured by any data. The keywords you enter can be used to search for this particle ID.
Visibility
Visibility, set by the person who enters the nanoparticle information, determines access to the nanoparticle. (Groups listed are user groups which have been defined in the caNanoLab security model.) Select one or more groups you wish to have access to the nanoparticle you are submitting.
Note: This option sets visibility requirements for the nanoparticle. You can set a different visibility when you submit assay results files, restricting access to the actual files.
To clear the entries and start again, click the Reset button.
3.
When you have finished defining the nanoparticle, click the Submit button. A message appears indicating the submission.
The Navigation Tree opens in the left hand panel of the browser, where you can add composition and characterization data for the nanoparticle sample you submitted. For more information, see Understanding the Navigation Tree.
