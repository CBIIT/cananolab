Nanoparticle Function Page
The form that opens from all Function subcategories is identical. Fields on the form are described in the following table. Note that the form described becomes extended when the “Update...” buttons are pressed. The additional fields that display at that point are also described in the following table.
Nanoparticle Function Fields
Definition
Functions Summary
 
Activation Method
Enter the method used to activate the nanoparticle.
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Particle Tree. The asterisk indicates a required field.
Description
Enter any additional functional information that the form does not include.
Linkage Information
 
Number of Linkages
Linkages link functional agents to the nanoparticle. Enter the number of particle linkages.
Update Linkages
Note: If you enter any number greater than “0”, click the Update Linkages button at the bottom of the page. This extends the page with the corresponding number of additional fields so you can enter additional information for each unit, as described below.

Agent Type
An agent is a functional molecule linked to the nanoparticle being annotated. Select the type of agent linked to the nanoparticle being annotated:
Peptide; Enter sequence
Small Molecule; Enter Name and Compound Name
Antibody; Enter Name and Species
DNA; Enter Sequence
Probe; Enter Name and Type
Image Contrast Agent; Enter Name and Type
Other; Enter agent type not included in the list.
Agent Description
Enter any additional agent information that the form does not include.
Number of Agent Targets
An agent target is an object fixed as a goal or point of examination or destination by the agent. Enter the number of Agent Targets.
Update Agent Targets
Note: If you enter any number greater than “0”, click the Update Agent Targets button at the bottom of the page. This extends the page with the corresponding number of additional fields so you can enter additional information for each unit, as described below.
