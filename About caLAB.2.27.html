Run Details - Workflow General Information
On the Search Run Results page, select the radio button for the appropriate run, then click the View Details button. The page that opens displays details associated with the corresponding run in the left panel (Run Tree), and general information about the associated workflow in the table on the right. The general information is described in the following table.
Workflow
Information Rows
Description
Source Name
The name of the institution which is the source of the sample(s) used in the run(s) that met the search criteria.
Assay Type
The assay type of the run search hit.
Assay Name
The name of the assay of which the run is a component.
Run Name
The name of each run for which the details display
From this page you can open input or output files associated with the workflow. For more information, see Uploading Input/Output Files.
The Run Tree is describe in the topic, Working With a caNanoLab Workflow.
