Displaying a Detail View of a Physical Characterization
If a Physical Characterization is added under a subcategory in the Navigation Tree, you can display a detail view of the characterization in a table. You can then print the table or save the table to a spreadsheet file. Each Physical Characterization subcategory is defined briefly:
Molecular Weight
Morphology
Purity
Shape
Size
Solubility
Surface
To display a detail view of an existing physical characterization, follow these steps:
1.
Under one of the Physical Characterization subcategories listed above, click a hyperlinked name in the Navigation Tree. The Physical Characterization page opens.
2.
The following table lists the details that appear for the characterization.
If the table is missing information or empty, the fields were not filled in when the characterization was added.
Field
Description
Description
The description of the characterization.
Protocol
The protocol from which the data is derived and/or the identifying version information for the protocol from which the data is derived. A hyperlink of the protocol text file may appear. Click the hyperlink to open or save the file.
Instrument
Identifies the kind of instrument used to obtain data, the instrument’s abbreviation, if any, and additional information.
Characterization File Description
The description of the characterization file which may be a graph, spreadsheet or image, or derived data. The characterization files are numbered in the order they were defined.
Characterization File
A link to or a graphic representation of each characterization file which may be a graph, spreadsheet or image, or derived data. The characterization files are numbered in the order they were defined.
To view the characterization file graphic, click the graphic in the table. The graphic opens in a new, larger window.
Derived Data
Lists the data derived from the characterization file. The characterization files are numbered in the order they were defined.
