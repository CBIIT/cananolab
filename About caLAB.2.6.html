caLAB Welcome Workflow Page
Once logged in on the caLAB Welcome Login Page, caLAB opens to a second Welcome page. The options available on this page depend upon the permissions granted when your user account was created.
All users can perform these tasks:
•
Access caLAB Quick Links (hypertext links in the left sidebar of the browser) that include Quick Links to the National Cancer Institute (NCI) website, the NCI Center for Bioinformatics website, and the Nanotechnology Characterization Laboratory (NCL) website.
•
Review a recommended caLAB workflow. For more information, see Executing a caLAB Workflow.
•
Use caLAB menu options, displayed according to assigned permissions. Available options which may be available are described as follows:
Menu Options
Description
Workflow
caLAB 1.0 has implemented one workflow: a reference workflow contributed by the Nanotechnology Characterization Laboratory (NCL). Click this option to execute the workflow. For more information, see Executing a caLAB Workflow.
Inventory
Select from the drop-down menu the name of the Sample Type to which the current sample belongs. Example: Dendrimer, Liposome.
Search
Click this option to open the search feature list, from which you can initiate searches of the database for Workflows, Samples, Aliquots and Nanoparticles. See Launching a caLAB Search for more information.
Nanoparticle Search
Click this option to initiates searches only for defined nanoparticles.
Home
Click this option to return to the caLAB login page. Clicking this button logs you out automatically.
Help
Click to open full help features. For more information see Using CGEMS Online Help.
Logout
Click to logout of caLAB.
