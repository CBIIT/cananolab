Submitting a Protocol
To submit a protocol, follow these steps:
1.
Select Submit > Protocol. This opens the Submit Protocol page.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Items with an asterisk are required fields.
Fields displaying [-?-] are editable, even after existing data populates the fields.
Search Run Fields
Description
Protocol Type*
Enter or select from the drop-down list the protocol type.
Protocol Name*
Enter or select from the drop-down list the protocol name.
Protocol Version
Enter or select from the drop-down list the protocol version number.
Protocol File
Click the Browse button to navigate to the protocol file.
File Title
Enter the title of the protocol file.
Description
Enter any additional description information that the form does not include.
Visiblity
Visibility, set by the person who enters the visibility information, determines access to the protocol for the selected particle ID. (Groups listed are user groups which have been defined in the caNanoLab security model.) Select one or more groups you wish to have access to the protocol you are defining.
3.
When you have finished, click Submit or click Reset to clear all fields on the form.
