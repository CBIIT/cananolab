Performing a Remote Nanoparticle Search
You can access this page after logging into caNanoLab or you can access publicly available nanoparticle report results from the Login page.
The Grid Search link on the Search Nanoparticles page opens the Remote Search for Nanoparticles page. With this page, you can search for publicly available nanoparticles from all active caNanoLab data services connected to the caBIG™ grid (caGrid™). For more information about caGrid, see https://cabig.nci.nih.gov/workspaces/Architecture/caGrid. An added benefit is that, not only are searches conducted on caGrid, but this feature publishes the caNanoLab domain model on caGrid for the benefit of other users of caNanoLab.
The Remote Search for Nanoparticles page lists the active grid node host names, which are discovered at run time, display. You can choose one or more of them to search. Grid node host name(s) also display in the result.
To launch this search, follow these steps:
1.
Click the Nanoparticles caNanoLab menu option. On the Manage Nanoparticle Home page, click the Search Existing Nanoparticles link. This opens the Search Nanoparticles form.
2.
Select the Grid Search option at the top of the page. This opens the Remote Search for Nanoparticles Search page.
3.
Enter text or select options as appropriate (described in the following table). Add an asterisk for a wildcard search.
If you click the Search button having defined no parameters, caNanoLab searches all remote caNanoLab nodes for available nanoparticles.
