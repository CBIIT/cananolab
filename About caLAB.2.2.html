About caNanoLab
Introduction to caNanoLab
Welcome to the cancer Laboratory Analysis Bench (caNanoLab). caNanoLab is a Laboratory Information Management System (LIMS) designed to facilitate data sharing in laboratories performing nanoparticle assays.
caNanoLab provides support and functionality for following features:
•
Support for the accessioning of samples (nanoparticles) and the creation of associated aliquots
•
Execution of nanoparticle assays
•
Recording of assay results
•
Search features for searching the objects in the caNanoLab database
•
Annotation of nanoparticles with characterizations
•
Sharing of these characterizations in a secure environment
caNanoLab source code is available at the National Cancer Institute Center for Bioinformatics (NCICB) download center: http://ncicb.nci.nih.gov/download/.
