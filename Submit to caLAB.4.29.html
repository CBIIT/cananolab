In Vitro Characterization Forms
In Vitro characterization is defined as the determination of the effect of nanoparticles on living cells in an artificial laboratory environment outside of the living organism.
Most of the fields on the forms that open for the In Vitro characterization categories are identical. They differ only when you click Number of Histogram/Bar Charts, enter the number and click Update. At that point, the page extends, and for different characterizations, the value names are slightly different, as noted in the following table.
In Vitro Characterization Field
Description
Characterization Source
Enter the source from which the characterization data comes, such as a vendor or a laboratory, or both.
View Title*
Enter a unique and meaningful name for the characterization. The title you enter will display on the Particle Tree. The asterisk indicates a required field.
Protocol Name
Enter the protocol from which the data is derived.
Protocol Version
Enter identifying version information for the protocol version from which the data is derived.
Description
Enter information not covered by other fields on the form.
Instrument Information
Instrument Type
Identify the kind of instrument used to obtain data.
Other Instrument Type
Enter Instrument Type that is not listed in the drop-down list.
Instrument Manufacturer
Select the manufacturer of the instrument used to obtain the data.
Other Manufacturer
Enter Other Instrument Manufacturer that is not listed in the drop-down list.
Description
Enter information about instrumentation not covered by other fields.
Histogram/Bar Chart Information
 
Number of Histogram/Bar Charts
If histogram(s) or bar chart(s) exist for the particle you are annotating, enter the number of them and click Update. This extends the page with the corresponding number of additional fields so you can enter information for each unit.
Histogram/Bar Chart Information Extension
 
Characterization File Name
This field remains empty until you select the file to be uploaded. To do so, click the Load File button, opening the Load Characterization File page. For more information about completing the form, see Loading Characterization Files.
{Characterization value name} Percentage
The value name for the in vitro characterization form is the last part of the form title. That value name is repeated here. Enter the percentage of toxicity here.
Is control?
Click Yes if there is a control for this characterization; click No if there is no control.
