Defining Function Information
To define function information for the functionalizing entity, follow these steps.
Click [Other] to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear.
1.
Click the Add button to extend the page where you can add function information.
2.
Select from the Function Type drop-down list the function type category which best fits the sample entity.
•
Imaging
•
Targeting
•
Therapeutic
3.
Description: Enter any additional composition information that the form does not include.
4.
When you have finished, click Submit or click Reset to clear all fields on the form.
In the Navigation Tree, a hypertext link for the new Composition characterization is added under Functionalizing Entity.
After defining the function Information, continue the process described in Adding Functionalizing Entity Composition Annotations.
