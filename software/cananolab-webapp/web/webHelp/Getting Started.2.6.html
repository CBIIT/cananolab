Welcome to caNanoLab Login Page
From the caNanoLab Login page, you can perform the following tasks:
•
Browse publicly available sample information. For more information, see Browsing Public Information.
•
Review current information about caNanoLab such as relevant statistics, features in the current application, functional and general frequently asked questions and see a count of visitors to the caNanoLab site.
•
Log in with an existing user account. For more information, see Logging into caNanoLab/Updating Password.
Before you can login, you must apply for an account to the caNanoLab Admin who will create the user account for you. You will be notified when the account is ready. caNanoLab validates your login and alerts you if you have used an incorrect user name or password. For more information, see Registering a New User.
Before you can log in, you must send an email request to NCICB Application Support. In the email subject, please specify the URL of the caNanoLab site to which you are applying the account.
•
Update the password for your user account. For more information, see Logging into caNanoLab/Updating Password.
To familiarize yourself with caNanoLab, the Welcome page provides the following options.
•
The What’s New on the right half of the page lists any recent changes to caNanoLab.
•
The Help menu option at the top of the browser accesses the complete caNanoLab Online Help project.
•
The Help link in the Login section of the page provides page-sensitive help topics.
