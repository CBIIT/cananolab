Creating New User Accounts
To configure new users, you must log into caNanoLab as an administrator.
Follow these steps to configure new users:
1.
Click the Administration menu bar option.
2.
In the page that opens, click the Create User Accounts and Assign Privileges link. This link leads to the NCI’s Common Security Module (CSM) where you can create users in the User Provisioning Tool (UPT). You must have administrative privileges to perform the tasks associated with creating a new user.
You can find more information about creating new users and assigning user privileges in the UPT User’s Guide, available at this site https://gforge.nci.nih.gov/docman/view.php/69/4728/calab_UPT_User_Guide.pdf.
