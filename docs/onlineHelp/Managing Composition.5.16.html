Adding a File
To add results in the form of supporting documentation to the nanomaterial entity, follow these steps.
Click [Other] to enter an alternative to selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear
1.
In the File section, click Add to expand the page where you can add file(s) to the nanomaterial entity.
2.
Select the Upload button, then click the Browse button to navigate to the report or publication to be associated with the sample,
– OR –
Click the Enter File URL button and enter the URL where the document is located.
3.
For the file to be added, specify or enter the following:
a.
File Type (required): Document, Graph, Image, Spreadsheet
b.
File Title (required)
c.
Keywords to associate with the file
d.
Description: Enter any additional information for the file that the form does not already include.
4.
In the Visibility field, select the group(s) to be granted access to the selected file.
5.
After specifying files, click Save to finish entering the data or Cancel to close the window.
You can add as many files as you wish.
After adding one or more files, return to the steps described in Adding Nanomaterial Entity Composition Annotations.
