Adding a File
To add results in the form of supporting documentation such as graphs, spreadsheet or images to the sample characterization, follow these steps. You can add as many files as you wish.
Click [Other] to enter an alternative to the selections listed. At that point, your entry is enclosed in [new entry] brackets, indicating that your entry is temporary. Once you click Submit (or Update), your entry is persisted and the brackets disappear.
1.
In the Finding segment of the page, click Add corresponding to File to extend the page.
2.
Select the Upload button, then click the Browse button to navigate to the report or publication to be associated with the sample,
– OR –
Click the Enter File URL button and enter the URL where the document is located.
3.
For the file to be added, specify or enter the following:
a.
File Type (required): Document, Graph, Image, Spreadsheet
b.
File Title (required)
c.
Keywords to associate with the file
d.
Description: Enter any additional information for the file that the form does not already include.
4.
In the Visibility field, select the group(s) to be granted access to the selected file.
5.
After specifying files, click Save at the lower right to finish entering the data or Cancel to close the window.
6.
Click Save in the Finding window, as well.
After adding one or more files, continue by Adding Data and Conditions or if you have no derived data to add, return to the steps described in Adding a Physico-Chemical Characterization.
