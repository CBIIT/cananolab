caNanoLab Welcome Main Page
Once logged in on the caNanoLab Welcome Login Page, caNanoLab opens to a second Welcome page. The options available on this page depend upon the permissions granted when your user account was created.
All users can perform these tasks:
•
Access caNanoLab Quick Links (hypertext links in the left sidebar of the browser) that include Quick Links to the National Cancer Institute (NCI) website, the NCI Center for Bioinformatics website, and the Nanotechnology Characterization Laboratory (NCL) website.
•
Use caNanoLab menu options, displayed according to assigned permissions. See Permissions and Visibility. Options which may be available are described as follows:
 
Menu Options
Description
Workflow
caNanoLab provides support for performing workflow searches or executing assays performed by researchers with assigned permissions. Users with permissions can click this option to execute a workflow. For more information, see Executing a caNanoLab Workflow.
Inventory
Under the Inventory menu option, caNanoLab supports the accessioning of samples and creation of aliquots. It also supports searches for samples and aliquots in the caNanoLab database. The Inventory option is available only to researchers with assigned permissions. Users with permissions can click this option to work with samples and aliquots. For more information, see Managing caNanoLab Inventory.
Search
caNanoLab allows logged in users to search nanoparticles and their characterizations, nanoparticle reports and protocols in the caNanoLab database. See Launching a caNanoLab Search for more information.
Remote Search
The Remote Search option allows users to auto-discover remote caNanoLab grid services and conduct remote searches for nanoparticles and reports against these remote caNanoLab grid services.
Submit
caNanoLab facilitates the submission of nanoparticle annotations (characterizations, assay results), reports and protocols. This feature is only available to PIs. Users with permissions can select one of these options to annotate nanoparticles, to publish a report or to submit a protocol. See Sample and Nanoparticle Definitions, Managing Protocols, or Submitting a Nanoparticle Report.
Administration
caNanoLab allows authorized administrators to manage user accounts, groups, roles, and protection elements. This feature is only available to users with assigned administrative permission.
Home
Click this option to return to the caNanoLab login page. Clicking this button logs you out automatically.
Help
Click to open full help features. For more information see Using caNanoLab Online Help.
Logout
Click to logout of caNanoLab.
The flowchart on the main page illustrated caNanoLab tasks and their functionality. For more information, see caNanoLab Functionality Overview.
