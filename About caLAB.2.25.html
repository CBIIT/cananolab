Searching Runs
This option in caNanoLab is part of the Execute Workflow feature, described in Executing a caNanoLab Workflow.
To launch a search for Runs to include in the workflow, follow these steps:
1.
From the caNanoLab menu options, select Workflow > Execute Workflow.
1.
In the Execute Workflow Home page, click Select an Existing Assay Run.
2.
On the Search Runs form, select or enter information for the following fields, as appropriate.
When defining search criteria, there are no "required" fields. You can specify only those parameters that you consider essential to the search. If you click the Search button having defined no parameters, compoÂ­nents of all of the caNanoLab runs will display in the Search Results..
Search Run Fields
Description
Sample Source
Select in the drop-down list the institution which is the source of the sample.
Assay Type
Select from Pre-Screening or In Vitro, the two assay types populated with data in caNanoLab 1.0.
Assay Name
When you select the Assay Type on this form, the corresponding subset of data is updated on this list. Select the name of the assay that is used in the workflow being searched.
Run By
Select in the drop-down menu the name of the person who physically ran this experiment in the lab. Anyone who has caNanoLab user account displays in this list.
Run Date
Click the calendar to select the date of the run.
3.
Click the Submit button OR click Reset to clear the form.
