caNanoLab Functionality Overview
After you have logged into caNanoLab, the application opens automatically to a Welcome page that displays the caNanoLab functions flowchart, represented in the figure in this topic.
NCL researchers can create samples and aliquots in the Inventory feature. Once samples are created, researchers can execute their assays within the laboratory Workflow by creating an assay run, specifying aliquots used within the run, and uploading associated input (sample sheet, instrument configuration) or output (assay result) files. The laboratory workflow acts as a repository for multiple experiment files for each assay execution (run).
Note: Workflows in caNanoLab must be incorporated into the system by the caNanoLab team; they become workflow templates. caNanoLab 1.1 has implemented one workflow: a reference workflow contributed by the Nanotechnology Characterization Laboratory (NCL). NCL researchers can use the workflow template to load assay input files, assign aliquots, and load output files (assay results).
After samples are accessioned into caNanoLab, NCL researchers can also Annotate a nanoparticle with characterizations and publish specific assay result files and reports. Characterizations and reports can be published to multiple user groups such as the public, CCNE researcher, etc. The annotation feature facilitates information sharing of characterizations and assay results outcomes
