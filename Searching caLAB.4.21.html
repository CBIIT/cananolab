Remote Nanoparticle Search
To launch this search, follow these steps:
1.
Select Remote Searches > Remote Search Nanoparticles.
2.
In the dialog box that opens, enter text or select options as appropriate (described in the following table). Add an asterisk for a wildcard search.
If you click the Search button having defined no parameters, caNanoLab searches all remote caNanoLab nodes for available nanoparticles.

Select from the drop-down list the category of function to which the nanoparticle contributes.
•

•

•

•

Characterization Type
Click from the displayed list on the left, the characterization type hypertext link that applies to the nanoparticle to be searched.
When you click a hypertext link, Characterization sub-categories display in the right-hand pane of the row. Select one or more of the items in the list. (CTRL + shift + click to select more than one.)
Grid Node Host
Select from the available remote nodes to query.
3.
Click Search to execute the search. To clear all fields, click the Reset button.
For information about search results, see Remote Nanoparticle Search Results. See also Remote Searches.
