Launching a Basic Search
1.
To launch a basic search for a sample, continue from step 2described in Searching Existing Samples.
2.
Enter text or select options as appropriate (described in the following table) in the search form. In any text field, you can add an asterisk for a wild-card search.
When defining search criteria, there are no "required" fields. You can add only those parameters that you consider essential to the search.
Sample Search Options
Description
Search Site
With this option, you can search for publicly available samples or private samples, with appropriate permissions and when you are logged in, from your local service and/or all active caNanoLab data services connected to the caBIG® grid (caGrid™). You can select multiple sites by using conventional selection techniques: ctrl + click, or shift + click.
Keywords
Enter one or more keywords by which samples could be identified in this search. caNanoLab can search for characterization or publication keywords, as well as text describing characterizations.

Enter text for the name or organization point of contact.

Note:
If you click the Search button having defined no parameters, all of the caNanoLab samples will be listed in the Search Results.
To clear all fields and start again, click the Reset button.
3.
To execute the search, click the Search button.
For information about search results, see Sample Search Results.
