Registering a New User
As a new user of caNanoLab, you can register online. From the Welcome to caNanoLab Login Page, click the Register link on the menu bar. In the page that opens, enter the appropriate information. Fields with an asterisk are required.
•
*Title and *Name
•
Contact information: *Email address and *Phone
•
List your *Organization
•
Enter a Description of your registration request, where appropriate
•
Select to be included in the caNanoLab user group email list
•
Click Register on the form to confirm your registration request.
Registered users are entered into the system by an administrator using the User Provisioning Tool (UPT). The administrator assigns all users to group “Public” when their user accounts are first created within the UPT tool. For more information, see Creating New User Accounts.
See Permissions and Visibility for more information.
