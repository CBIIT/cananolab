Adding Data and Conditions
In the Data and Conditions segment of the window, you can create a matrix where you can enter data values and/or other information that are part of your findings.
To define the matrix for the data, follow these steps:
1.
To define the matrix size, enter numbers for rows and columns and click the Update button.
The window now displays column [#} headings that you can name for your matrix.
2.
Click on the underlined column heading(s). Available options can be specific for the type of characteristic you are defining. As described below, select an option or if there are none or if you prefer, select [Other] to name the column yourself.
a.
Column Type – You must select either Datum or Condition. In the cases noted below, based on your Characterization selection at the top of the Characerization page, customized options may display for the following fields.
b.
Column Name and Column Property – Select or enter a column name.
•
If you selected Datum for Column Type, click [Other] to name the column yourself.
•
If you selected Condition for Column Type, for all characterizations, Column Name displays the options listed in the left column of the following table. When you make one of these name selections, Condition Properties options may correspond to the name you selected; those are shown in the right column. After specifying a name, select one of the column properties, where appropriate.

e.

3.
Click Set Column Order to indicate the order of the column headings in the matrix you have defined.
4.
Click Save to confirm the your matrix design choices. After you do so, the names you selected show as column headings. Click Remove to delete the column, or Cancel to close the window.
5.
Click Save in the Finding window, as well.
After adding data and conditions to the sample, continue the process described in Adding an In Vivo Characterization.
