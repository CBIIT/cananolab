Characterization Summary View
In caNanoLab Nanoparticle Characterization is defined with Physical and In Vitro Characterizations. Molecular Weight is an example of a Physical Characterization category. Oxidative Stress is an example of an In Vitro Characterization.
If there are multiple characterizations added under a subcategory in the Navigation Tree, you can display a summary view of all the characterizations in one table. You can then print the table or save the table to a spreadsheet file.
To display all the characterizations in one table, follow these steps. (Asterisks indicate required fields.)
1.
Click the subcategory name itself, such as Physical Characterizations > Molecular Weight or In Vitro Characterizations > Oxidative Stress. The Physical or In Vitro Characterization - [subcategory] page opens listing all the characterizations in one table, one characterization per row.
The following table lists and describes the fields that appear in a summary for each physicalm or in vitro characterization that has multiple entries.

Description
View Title
A unique and meaningful name for the characterization. To display a detail view of the characterization, see Characterization Details View.
Characterization File
A link to or a graphic representation of each characterization file which may be a graph, spreadsheet or image, or derived data. The characterization files are numbered in the order they were defined.
To view the characterization file graphic, click the graphic in the table. The graphic opens in a new, larger window.
Instrument Info
The instrument used to obtain data, the abbreviation of the instrument, and the manufacturer of the instrument.
