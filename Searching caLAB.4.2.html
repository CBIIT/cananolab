Launching a caNanoLab Search
Access to features and objects in caNanoLab is dependent upon the permissions assigned registered users. See Permissions and Visibility. The caNanoLab 1.2 allows you, with permissions, to perform the following searches:
Run Search
A Run Search, to identify Runs that can be included in a Workflow, is part of executing a Workflow. See Searching Runs.
Workflow Search
A Workflow Search looks for Run information such as a file, an aliquot, etc. See Searching a Workflow
Sample Search
A Sample Search, initiated under the Inventory menu option, looks for samples that may be recorded in caNanoLab. Samples are identified by source vendors, date of receipt, storage location, etc. See Searching Samples.
Aliquot Search
An Aliquot Search, initiated under the Inventory menu option, looks for aliquots, identified by the source from which the aliquot is derived, the name of the person who submitted the aliquot, storage location, etc. See Searching Aliquots.
Nanoparticle Search
A Nanoparticle search looks for a nanoparticle based on annotations that have been added or characterization or data files that have been uploaded. An advanced nanoparticle search allows the user to specify advanced characterization information for the search. See Searching Nanoparticles
Reports Search
A Reports search looks for submitted reports associated with multiple nanoparticles. This feature is accessible even to users for whom access to nanoparticles in caNanoLab is restricted. See Searching Nanoparticle Reports.
Remote Searches
These searches display all active grid node host names connected to the caBIG™ grid (caGrid™), discovered at run time, which you can select for a remote nanoparticle search or a remote nanoparticle reports search. Grid node host names display as part of search results. See Remote Searches.
