Publishing a Nanoparticle Report
This feature allows you to upload a final nanoparticle report for a vendor or researcher. The report can contain information about multiple particles.
Note:
This page can be opened from two places in caNanoLab 1.1:
1.
If you want to upload a nanoparticle report for a particular vendor or researcher, select Submit > Publish Report.
2.
After you perform a nanoparticle report search, on the Nanoparticle Reports Search Results page, click the hypertext link in the Report Title column. This Submit Nanoparticle Report page opens, allowing you to review the report details. For more information, see Nanoparticle Report Details Page.
To publish a report, follow these steps:
1.
Click Submit > Publish Report.
2.
On the Submit Nanoparticle Report form that opens, select or complete the folÂ­lowing fields:
Nanoparticle Report Criteria
Description
Particle ID
Enter the Particle ID, the subject of the report
Report Type
Select the report type from the drop-down menu.
Report File
Browse for the file you want to upload.
Report File Title
Enter the report title.
Report File Description
Enter information not covered by other fields on the form.
Visibility
The term, Visibility, is used to describe the access to objects granted to logged in users. Select the group to which this report is to be visible.
3.
Once you have entered the appropriate information on the form, click Submit.
OR
Click Reset to clear the fields on the form.
