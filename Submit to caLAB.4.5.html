Particle Tree
The Particle Tree displays nanoparticle annotation information for the selected nanoparticle in the left-hand panel of the browser. Initially only the particle type and ID you have selected in the General Information page display. The hypertext link on the Particle Tree takes you back to the General Information page that just closed.
If other annotation information has already been filled out for the nanoparticle, it displays in abbreviated form on the Particle Tree.
•
To add to, or edit the annotations, click the Add button to the right of the Particle Tree categories.
•
Click the hypertext link to open annotation details.
Note:
If you cannot edit or add information to a particular category, the Add button to the right of the categories is hidden.
