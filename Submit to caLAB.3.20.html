Loading Characterization Files
When defining Physical Characterization, many of the characterization forms include one of the following options: Update Distributions, Update Images, Update Plot Charts, or Update Histograms. These options generally refer to a distribution graph, but it is dependent up the type of characterizations, for example Size Distribution by Volume or Size Distribution by Intensity.
If, on the form, you enter a number greater than “0” for Number of {graphs}, then to appropriately complete the nanoparticle annotation, you should click the Update {graphs} button. This extends the page where you can enter additional {graph} information and upload a corresponding file. To upload a file, click the Load File button.
In the Load Characterization form that opens, complete the information, as described in the following table:
Load Characterization Fields
Definition
Select File
Select one of these options:
•
Choose File from caNanoLab workflow. With this selection, you choose a file already uploaded into caNanoLab via the caNanoLab workflow function. See Working with Files.
•
Upload New File. With this selection, browse for the new file associated with these characterizations.
Assay Result File Title*
Enter a unique and meaningful name for the file title. The title you enter will display on the Particle Tree. The asterisk indicates a required field.
Assay Result File Description
Enter a description for the Assay Result file.
Comments
Enter any additional description comments that the form does not include.
Keywords
In the Keywords text field, add any keyword you want associated with the selected characterization file, one keyword per line. You can enter something not necessarily captured by any data. The keywords you enter can be used to search for this particle ID.
Visibility
The Visibility option, set by the person who enters the annotation information, determines access to the characterization file to be uploaded. (Groups listed are user groups which have been predefined in the caNanoLab security model.) Select one or more groups you wish to have access to the file you are uploading.
 
Note: This visibility setting does not override the visibility you set on the Nanoparticle General Information page.
