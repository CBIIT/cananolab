Annotating a Nanoparticle
To annotate a nanoparticle, select Submit > Annotate Nanoparticle, which opens an Annotate Nanoparticle summary page. This page describes a nanoparticle and offers two links for working with nanoparticles.
Annotate a New Nanoparticle
This link opens a form for annotating a nanoparticle in the caLAB database to which no annotations have yet been added. For more information, see Nanoparticle General Information.
Select an Existing Nanoparticle
This opens the form for launching a search for an existing nanoparticle to which annotations have already been added. For more information, see Searching Nanoparticles.
Note:
It is possible to have multiple annotation for one particle, coming from different sources.
