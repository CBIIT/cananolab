Documents Category
When you click the Navigation Tree Documents button, all reports and/or publications that were added to the nanoparticle using the Documents menu option expand under the Documents section. (Click the button again to collapse the report names).
•
To access a document, click the hypertext name. This opens the document.
•
To add a document to the open nanoparticle, click the Add button. This opens the Submit Publication page identical to that opened from the Documents menu option with one exception:
If you work from the Documents menu option, you can associate multiple nanoparticles with a publication or report. If you add documents to an open nanoparticle, you can use the copy feature to associate the document with other nanoparticles.
For more information about adding a publication, see Submitting a Publication.
•
For information about document details, see Searching Nanoparticle Documents and other document topics in the Managing Documents section.
