Composition File
This section of the Sample Composition sidebar options allows you to add files relevant to the nanoparticl sample independently of the entity or chemical associations that you have defined. You can add as many files as you wish. To add a file, follow these steps:
1.
Click the Composition File button in the Sample Composition section of the left sidebar.
2.
Fill in each of the following fields on the Composition File Information form:
Composition File Fields
Definition
Upload Report File
or
Enter Report URL
Click the Browse button to navigate to the file to be associated with the report, or enter the URL where the report is located.
File Type*
Select the file category from the drop-down menu.
File Title*
Enter the report title.
Keywords
Keyword(s) you identify to associate with the submitted nanoparticle.
Visibility
The group(s) which have access to the selected nanoparticle.
