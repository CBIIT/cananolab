Search Run Results
Run Search Results display in the browser window in table format. Column headings are appropriate for the sample type for which you searched; they are described in the following table:
Search Run
Results Columns
Description
Select
Select one of the search hits for which you want to view details. Click the View Details button for detailed information about the runs.
Sample Source
The institution which is the source of the sample(s) used in the run(s) that meet the search criteria.
Assay Type
The assay type of the search hit.
Assay Name
The name of the assay of which the run is a component.
Run Name
The name of each run that meets the search criteria
Run By*
The name of the person who physically ran this experiment in the lab.
Run Date*
The date of the run.
Aliquots
The IDs of component aliquots of the run
For any hit for which you want more information, select its corresponding radio button in the first column. Click the View Details button. This opens a page that displays General Information for the workflow associated with the corresponding run, as well as a corresponding Run Tree on the left. See Run Details - Workflow General Information for more information.
For information about launching a Run Search, see Searching Runs.
