Nanoparticle Report Details Page
The Report File page opens when you click any of the hypertext file name links in the Nanoparticle Reports Search Results page.
The page that displays is the same Report File page on which the report was submitted into caNanoLab. Details are described in the following table.
Nanoparticle Report Columns
Description
Report File Type
The report type.
File Name
The name assigned by caLAB to the file.
File Title
The report title assigned when the report was submitted.
File Description
The description of the file.
Visibility
The group to which this report is visible.
