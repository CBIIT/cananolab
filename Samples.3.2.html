Samples Overview
When you click the Samples option on the caNanoLab menu bar, the Manage Sample/Aliquot Home page appears. The Sample Links and Aliquot Links panels enable you to create new and search for existing samples (see Managing Samples) and create, edit, an search for existing aliquots (see Managing Aliquots).
For more information about samples, see Description of a Sample.
