Registering a New User
The caLAB Welcome Login Page provides the means whereby you can register a new user of caLAB. To do so, follow these steps:
1.
Just below the user login text boxes, click the Register hypertext link.
2.
In the User Registration dialog box that opens, complete the registration form. Fields are self-explanatory. Items with an asterisk are required fields.
3.
After entering the new user information, enter the login ID and password.
4.
Click Submit to finalize the new user registration.
5.
To clear the registration information from the form, click Reset.
