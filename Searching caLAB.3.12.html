Nanoparticle Search Results
Note:
If you click the Search button having defined no parameters, all of the caLAB Nanoparticles will display in the search results.
Nanoparticle Search Results display in the browser window in table format. Column headings are appropriate for the nanoparticle for which you searched. They are described in the following table.
Nanoparticle Search Results Columns
Description
Particle ID
The fully qualified name of the nanoparticle to be searched.
Particle Source
The name of the Nanoparticle Source.
Particle Classification
The name of the Nanoparticle classification. Example: Organic:Hydrocarbon.
Particle Type
The name of the nanoparticle category to which the nanoparticle belongs. Example: Dendrimer
Particle Functions
A list displaying functionsthat have been entered for the hit Particle ID.
Particle Characterizations
A list displaying characterization annotations that have been entered for the hit Particle ID.
Particle Keywords
Keywords that can have been entered for the hit nanoparticle.
For any hit, click on a hypertext link in the Particle ID column (first) to open detailed information about the search hits. caLAB opens to the Nanoparticle General Information page where you can review nanoparticle details. Details for only one nanoparticle display at a time.
For information about configuring a nanoparticle search, see Searching Nanoparticles.
